# 犬が寝る前にくるくる回る本当の理由

チャンネル: ペット (pet) ／ 予測: high

## 手順

1. `grok_agent_prompt.txt` の中身を丸ごと Grok Agent Mode に貼る → 画像生成〜動画クリップ化まで一括で実行される
2. `narration.txt` を Grok Companion に読み上げさせ、音声を抽出する
3. CapCut にクリップ（カット番号順）+ ナレーション音声を読み込み、下記チェックリストの順に編集する
4. 「概要欄・投稿メタ」をコピペして YouTube Studio で予約投稿する

## フック（冒頭2秒・CapCut で最初に入れるテロップ）

その行動、野生時代の名残です

## CapCut 編集チェックリスト

- [ ] クリップをカット番号順に並べる（1カット目=フックに対応）
- [ ] ナレーション音声を全体にオーバーレイし、カットの切り替えを台詞の区切りに合わせる
- [ ] 冒頭にフックのテロップを大きく表示（最初の2秒で離脱させない）
- [ ] 自動字幕 or 手動キャプションを全編に付ける（無音視聴対応）
- [ ] BGM を追加（音量はナレーションの邪魔にならない程度に下げる）
- [ ] 締めに「他の犬の不思議行動も毎日紹介しています」のテロップ/ナレーションを確認
- [ ] 縦動画 1080x1920・尺 60秒以内で書き出し

## 概要欄・投稿メタ

**タイトル**: 犬が寝る前にくるくる回る本当の理由

**ハッシュタグ**: #犬のいる生活 #ペット雑学 #犬あるある

**アフィリエイトリンクなし**（価値提供ネタ。登録・保存を促すのみ）
