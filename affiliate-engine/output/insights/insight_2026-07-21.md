# インサイトレポート 2026-07-21

## 婚活

**市場トレンド（Web検索による）**
- 「婚活あるある」を列挙し、途中で気づき・格言的な一文に転換する構成が伸びやすい（例：条件チェックが『審査モード』化する話）
- 電車やカフェなど日常の中で見かけた『婚活中らしき2人』を観察し、距離感や会話のトーンから心理を考察する目撃エピソード型が定着
- 結婚相談所の『わきまえ』的な指導や、女性が違和感を飲み込んでしまう構造への気づき・言語化が共感を集めている
- 『負けたくない気持ち』『とりあえず同棲』など婚活・恋愛におけるマインドの思い込みを指摘し、問いかけで締めるメンタル系投稿が反応を得やすい
- 率直な自己紹介・年齢公開型の投稿も一定の反応があるが、投げかけ要素が薄いとエンゲージメントは伸びにくい傾向

**自アカウント実測の所見**: 現時点で投稿実績データ（サンプル数0件）が存在しないため、伸びた投稿・伸びなかった投稿の傾向比較はできません。まずは複数パターンを投稿し、閲覧数・保存数・返信数を記録できる状態を作ることが先決です。

**改善アクション**
- [ ] 直近のバズ傾向にある『あるある→気づき→問いかけ』構成と『観察・目撃→考察→問いかけ』構成をテンプレとして試験投稿し、初期データを蓄積する
- [ ] リンク入り投稿は全体の中で最大1本に抑え、価値提供型・問いかけ型を優先して投稿数を確保する（フォロワー獲得優先フェーズのため）
- [ ] 投稿ごとに『閲覧数』『保存数』『返信数』を必ず記録し、次回分析時に比較できるようにする
- [ ] 誇大表現・断定表現・体験談の捏造を避けつつ、一般論や観察ベースの文体で共感ラインを作る

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 副業

**市場トレンド（Web検索による）**
- ユーザー参加型の『質問形式』投稿はリーチが伸びやすく、シンプルな問いかけが高反応の共通パターンになっている
- 投稿にリンクを貼るとリアクションが大きく落ちる傾向があり、フォロワー獲得目的の投稿ではリンクなしが無難とされている
- 『あなたはどっち派？』のような二択・参加型の問いかけは、企業アカウントでも双方向コミュニケーションの定番手法として使われている
- 完璧に整えた発信より、人間味のある等身大の言葉の方が共感を呼びやすいプラットフォーム特性がある
- いいね・コメントが増えるほど伸びやすく、交流（返信・リポスト）をしているアカウントほど評価されやすい傾向がある

**自アカウント実測の所見**: 3投稿とも一定のview数は獲得できているが、いいねは全て0で『見られてはいるが反応されていない』状態。実用情報を端的にまとめた確定申告系の投稿が最も伸びており、知らないと困る系の実務ネタとの相性が良い一方、質問形式の投稿は自由回答型で答えにくく、期待したほど反応を引き出せていない可能性がある。

**改善アクション**
- [ ] 質問投稿は自由回答ではなく『二択・三択』形式に変えてリプライのハードルを下げる
- [ ] 冒頭一文を数字や意外性のあるフックにして、スクロールを止める力を強化する
- [ ] 確定申告系のような実務・制度情報の投稿を、チェックリスト形式（絵文字箇条書き）でさらに保存されやすい構成に統一する
- [ ] リンク投稿は1本のみに絞り、価値提供の文脈内に自然に組み込み、投稿後は他アカウントへの返信・交流を並行して行う

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 美容

**市場トレンド（Web検索による）**
- Threadsは2026年にかけて「会話量」を重視するアルゴリズムに大きくシフトしており、<cite index="11-1">「いいねの数よりも、「どれだけリプライがつき、その会話が続いたか」が評価されます」</cite>という傾向が強まっている
- 伸びる投稿の型として<cite index="16-1">「最初にお客様の悩みを提示することが伸びる動画の大前提になります」</cite>という「お悩みフック」構造が美容ジャンルでも定番化している
- テキスト単体でも<cite index="12-1,12-2">最初の一文で興味を引けるかどうかが勝負であり、「知らないと損する」「3分でできる」など読むと得する言葉が反応されやすい</cite>傾向が続いている
- 共感・本音系の短文投稿が保存されやすく、<cite index="15-1">「Threadsは画像より言葉のほうが圧倒的に伸びやすい」</cite>とされ、日常の本音を綴る投稿が会話を生みやすい
- 美容アカウントでは<cite index="8-1">「乾燥肌対策の裏ワザ」「自宅でできる簡単ヘアアレンジ」など、ターゲットの肌や髪の悩みに寄り添う有益な情報</cite>や質問投げかけ企画が交流を活発化させる切り口として使われている

**自アカウント実測の所見**: 毛穴の乾燥/皮脂タイプ質問投稿とクレンジング見直し投稿はどちらも閲覧1000超・いいね2件と、保存されそうな「お悩み提示＋一般論」の構造自体は機能しているが、いいね/リプライ転換率が低く「会話の連鎖」までは生まれていない。質問文が単発の二択で終わっており、リプライへの返信誘導や続きを促す一言が不足しているため、アルゴリズムが評価する「会話継続」のシグナルを取りこぼしている可能性が高い。

**改善アクション**
- [ ] 冒頭一文に「お悩みフック」＋意外性ワード（「実は」「知らないと損」等の誇張にならない範囲の軽い煽り）を必ず入れ、離脱を防ぐ
- [ ] 質問系投稿の末尾に「よかったら理由も教えてください」等、リプライへの再返信を誘発する一言を追加し、会話ラリーを意図的に設計する
- [ ] 保存需要が高い「リスト形式」「季節ネタ×手順の見直し」など、価値提供型の構造を増やし週内で複数パターンをABテストする
- [ ] 投稿時間を7〜9時／12〜14時／21〜24時のゴールデンタイムに合わせ、投稿直後30分はリプライ対応に集中して初速を作る

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
