# インサイトレポート 2026-07-07

## 婚活

**市場トレンド（Web検索による）**
- <cite index="1-1">婚活を進めるうちに年収・職種・実家などのチェック項目ばかりになり、気づけば「採用面接」のようになっている、という比喩でのあるある投稿が支持されている</cite>
- <cite index="7-1,7-2">電車やカフェなどで見かけた婚活中と思しきカップルの様子を第三者目線で観察・考察するタイプの投稿が反応を集めている</cite>
- <cite index="6-1">婚活や恋愛で「負けたくない」という意識が強いほどうまくいかないという心理的な逆説を提示する投稿が伸びている</cite>
- 同棲や共同生活の「意味の受け取り方」の男女差など、価値観のズレを構造化して見せる切り口が支持されている
- 元交際相手や婚活中のエピソードをシリーズ的に振り返り、共感とコメントを誘発する自己開示ストーリー型が根強い

**自アカウント実測の所見**: 自アカウントでは、短い一般論＋共感フレーズ（婚活疲れ＝不採用通知感覚）と、具体的な年齢・シーンを伴う感情ストーリー（32〜34歳の焦り、深夜の孤独）が上位に来ている一方、同じ感情ストーリー型でも冒頭のインパクトが弱いもの（夜中に泣いた、100人と会った系）は下位に沈んでおり、単なる自己開示だけでは差別化できていない。また価値提供Tips単体（休日の過ごし方）は中間的な結果にとどまり、独自の比喩や問いかけが伴わないと伸びにくい傾向が見える。

**改善アクション**
- [ ] 婚活を「採用面接」「審査」などのメタファーで構造化し、最後に意外性のある気づき・逆説で締める投稿を増やす
- [ ] 自己開示ストーリー型は継続しつつ、冒頭1文目に具体的な時刻・場所・数字を入れてインパクトを強化する
- [ ] 第三者観察型（電車・カフェで見かけた婚活中の人）の新フォーマットを追加し、共感と保存を狙う
- [ ] 価値提供Tips単体の投稿には必ず「問いかけ」または「比喩」を添え、返信・保存を誘発する構造にする

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 副業

**市場トレンド（Web検索による）**
- リンク添付投稿はエンゲージメントが大きく低下する傾向があり、フォロワー獲得目的の投稿ではリンクなしが基本という認識が広がっている
- Threadsのアルゴリズムは「会話の活発さ」を重視しており、投稿全体の約半分がリプライで構成されるほど返信を誘発する投稿が拡散されやすい
- 2026年は単発バズよりも『なぜその結論に至ったか』『何を捨てたか』といった思考プロセスを見せる投稿が差別化要因になっている
- 『後で見返したい』と感じさせる保存性の高い構成（動画×解説テキストや手順の言語化）が長期的な資産型コンテンツとして評価されている
- 露骨な『いいねして』『コメントして』のようなエンゲージメントベイトはMeta側が取り締まり対象にしており、自然な問いかけへの転換が必要とされている

**自アカウント実測の所見**: 現時点で自アカウントの投稿実績データ（sample_size=0）が計測されておらず、伸びた/伸びなかった投稿の比較分析ができない状態です。まずはインサイト機能を使った定量データの蓄積が急務であり、それまでは市場トレンドをベースに仮説検証型で投稿設計を行う必要があります。

**改善アクション**
- [ ] Threadsインサイトで閲覧数・インタラクション・フォロワー推移を毎投稿後に記録し、次回分析ができるデータベースを最低10〜15投稿分蓄積する
- [ ] 認知拡大フェーズはリンクなし投稿を8割以上とし、保存トリガー型（手順・チェックリスト化）とリプライ誘発型（軽い自己開示＋問いかけ）を交互に投下する
- [ ] 『正解だけ』を提示する投稿ではなく、選択に迷った過程や比較検討の視点を一文添えて思考プロセス型に寄せる
- [ ] 投稿後1時間はリプライに丁寧に反応し会話を継続させることで、拡散評価を得やすい状態を作る

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 美容

**市場トレンド（Web検索による）**
- Threadsは「拡散」より「共感」を重視するSNSとされ、悩みに寄り添う投稿や人柄が伝わる観察系の内容が評価されやすい
- 投稿の伸びは最初の一文で決まりやすく、数字や意外性を入れた一言フック（『知らないと損する』『たった一つ』など）が反応を集めやすい傾向がある
- 『乾燥肌対策の裏ワザ』などの悩み解決系Tipsや、質問を投げかけて交流を促す投稿（『今月のキレイの目標は？』等）がエンゲージメントを生みやすい
- 2026年美容トレンドとしては韓国美容の定番化、Y2K・バレエコア、レイヤーカット、パーソナルカラー診断、眉毛サロンなど『自分に合うものを見つける』系の話題が検索上昇中
- 情報が整理された比較・リスト形式（『〇〇5選』『比較画像』）は保存されやすく、テキストのみでも伸びるアカウントが増えている

**自アカウント実測の所見**: サンプル数1件のためまだ傾向断定はできないが、日焼け止めの『塗る量』Tips投稿は474viewsに対し保存されやすいTips構造ではあったものの、いいねは2件と反応率が低く、冒頭フックの強さや読者に行動（返信・保存）を促す一言が不足していた可能性がある。今後は同じ実用Tips路線を活かしつつ、質問締めや数字訴求を加えて反応率を検証する必要がある。

**改善アクション**
- [ ] 冒頭一文に『多くの人が知らない数字・意外性』を入れて離脱を防ぐフックを強化する
- [ ] 投稿の締めに『あなたは？』『試したことある？』などの質問を必ず1つ入れ、リプライ率を検証する
- [ ] 日焼け止めTipsのような実用小ネタを『比較・リスト形式』に発展させ保存率を狙う（例：順番・タイミングの小ネタ）
- [ ] リンク付き投稿は1本に絞り、残りは価値提供＋質問締めで認知拡大フェーズのKPI（フォロワー・保存・返信）に集中する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 教育

**市場トレンド（Web検索による）**
- 教育実習生や教員のSNS投稿を巡るネットリテラシー議論が直近2〜3週間で話題化し、『子どもの写真をどこまで公開するか』という線引きへの関心が高い
- Threadsのアルゴリズムは『返信率』を最重視する傾向が強く、質問・問いかけで終わる投稿が拡散されやすい
- Tips・比較・リスト形式でまとめた『保存したくなる』投稿は保存率・エンゲージメント率が高くなる傾向がある
- 『常識を覆すデータ』や『意外性のある事実』を冒頭に置く投稿は拡散力が高い
- プログラミング教育・知育玩具・幼児教育など『家庭でどう補うか』という切り口のコンテンツが継続的に関心を集めている

**自アカウント実測の所見**: サンプルが1件のみのため断定は難しいが、平均483ビューに対しいいねが1件と反応率が低く、投稿末尾に質問や問いかけがないため『読んで終わり』になりやすい構造だったと考えられる。観察・気づきの切り口自体はトレンドと合致しているため、締め方（CTA）の改善余地が大きい。

**改善アクション**
- [ ] 投稿末尾に必ず『問いかけ』を入れ、返信を誘発する構造に変更する（アルゴリズムが返信率を最重視するため）
- [ ] 単発の気づきだけでなく、Tips・比較リスト形式（2〜4項目）を組み込み保存率を高める
- [ ] 教育実習・SNSリテラシーなど直近の話題性が高いテーマを、個人の実体験ではなく一般的な観察・問題提起として扱う
- [ ] 投稿頻度を上げ、朝（7-9時）・昼（12-14時）・夜（21-24時）など反応が出やすい時間帯で複数投稿し継続性シグナルを強化する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
