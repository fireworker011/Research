# インサイトレポート 2026-07-16

## 筋トレ

**市場トレンド（Web検索による）**
- 感情を揺さぶる一言（『正直しんどい』『納得いかない』等）で注意を引く『感情フック型』が拡散に効果的とされている
- 『あなたはどう思いますか？』のように答えやすい質問を投げかけ、コメントを生みやすくする『問いかけ型』が伸びやすい型として紹介されている
- 日常の出来事に自分の価値観を乗せる『共感＋価値観提供型』、常識をひっくり返す『逆説型』が拡散を狙う型として整理されている
- 本文で興味を引き、コメント欄に深掘り情報を追加する『二段構え』構成により、スクロール→コメント欄への移動が生まれ、滞在時間とエンゲージメントが伸びる設計が広がっている
- Threadsは寄り添い系・悩み共感ワードが伸びやすい媒体特性があるとされ、悩みへの共感投稿がジャンル問わず支持されやすい

**自アカウント実測の所見**: サンプルが1件のみのため断定はできないが、視聴数1321に対しいいねが1と反応率が低く、共感型の内容自体は悪くないものの、読了後に『会話』や『保存』へつながる仕掛け（明確な問いかけや箇条書きでの情報整理）が不足していた可能性がある。文末が絵文字のみで締められており、コメントを誘発する導線が弱かったことも一因と考えられる。

**改善アクション**
- [ ] 冒頭一文に『感情フック』または『逆説』を入れて、スクロールを止める力を強化する
- [ ] 文末は絵文字だけで締めず、答えやすい具体的な質問で終えてコメントを誘発する
- [ ] 箇条書き＋絵文字で情報を区切り、保存したくなる『価値提供』の型を明確にする
- [ ] 本文で興味を引き、コメント欄に深掘り情報を追加する『二段構え』構成を試し、エンゲージメントの底上げを図る

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 教育

**市場トレンド（Web検索による）**
- Threads全体で「いいね」より「コメント（会話）」が評価軸の中心となっており、教育ジャンルでも問いかけ型の投稿がフォロー転換率を高めやすい
- 感情フック型（『正直しんどい』『納得いかない』等の一言で注意を引く）の冒頭が伸びやすい構造として定着
- 常識をひっくり返す逆説型（『〇〇と言われるが、実は逆』）が興味を引くフックとして機能している
- 自己開示や個人的ストーリーより、日常の出来事に価値観を乗せる『共感＋気づき提供型』が教育・子育て界隈で保存されやすい
- 投稿直後（15分〜1時間）の返信対応が拡散の鍵とされ、質問で締める投稿ほど会話が連鎖しやすい

**自アカウント実測の所見**: 自アカウントの上位投稿（送迎の負担、宿題の任せ方、やめる基準）はいずれも『親が直面する具体的な選定基準』を提示する観察型で、宿題の投稿は明確な問いかけで締めているため views が伸びている。一方、下位投稿は『しつけの設計』『文系でもプログラミング』など抽象的・教訓的な断定文で終わっており、読者が反応する余地（問い・共感の引っかかり）が少ない。全体を通していいね数が1〜3と極端に少なく、views規模に対してエンゲージメントが薄いため、保存はされても会話や反応には繋がっていない。

**改善アクション**
- [ ] 全投稿の末尾に『あなたはどうしていますか？』型の答えやすい問いかけを必ず入れ、コメント発生率を上げる
- [ ] 下位投稿のような抽象的・断定的な結論文（〜という問題です）を減らし、上位投稿のような『具体的な選定基準・判断軸』の提示に寄せる
- [ ] 冒頭1文を感情フック型または逆説型（『〜と思われがちだが、実は』）に変え、スクロール停止率を高める
- [ ] 投稿後1時間は返信対応を意識し、コメントへの即レスで会話の連鎖を作る（フォロワー外への露出増加を狙う）

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 節約

**市場トレンド（Web検索による）**
- 感情フック型の書き出し（『正直しんどい』『地味にきつい』等）で共感を先に取り、その後に具体策を出す構造が伸びやすい
- 断定せず『あなたはどう思う？』『どっち派？』のように誰でも答えやすい二択・簡単な問いかけで返信を誘発する型が支持されている
- 『量より質と言われるが実は〜』のような常識を一度ひっくり返してから本題に入る逆説型の冒頭が注意を引きやすい
- 画像なしのテキストのみでも保存・会話が生まれれば伸びる傾向があり、リスト形式（◯選・手順型）は保存されやすい
- 投稿直後のリプライへの即時返信が『会話の深さ』としてアルゴリズム評価されるため、質問で締めて会話を継続させる設計が重要

**自アカウント実測の所見**: 現時点でサンプル数0のため、伸びた投稿・伸びなかった投稿の傾向を統計的に比較できる実測データが不足している。まずは市場トレンドを参考にした複数パターンを投稿し、初期データ（views・保存・返信数）を蓄積することが優先課題となる。

**改善アクション**
- [ ] 感情フック型・問いかけ型・逆説型の3パターンを最低でも各3投稿ずつ試し、どの型が自アカウントで反応を得やすいか初期データを収集する
- [ ] リンク投稿は認知拡大フェーズの方針通り全体の1本以下に抑え、残りは保存・返信を狙う価値提供型に振り切る
- [ ] 投稿後1時間は返信に対応し、コメントへの反応を『会話の深さ』として蓄積し次回投稿の改善材料にする
- [ ] 節約ジャンルの具体トピック（固定費・食費・サブスク・ポイ活等）ごとにどの切り口が反応良いか記録し、伸びた話題は横展開する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
