# インサイトレポート 2026-07-10

## 婚活

**市場トレンド（Web検索による）**
- 「婚活が採用面接・審査モードになってしまう」というメタファー型のあるある投稿が共感を集めている
- 相手の言動を「レッドフラッグ／グリーンフラッグ」に分類するSNS発の評価軸が急速に浸透しトレンドワード化している
- 「タイパ（タイムパフォーマンス）婚活」など、無駄を減らし早期に対面へ進むスタイルが支持されている
- ファーストメッセージは『プロフィールを読んだ感想＋質問1つ』が黄金パターンとされ、コピペ的な文面は嫌われる傾向
- 親が代わりに相手を探す『代理婚活』など、当事者だけでなく周農の視点を絡めた新しい切り口が話題化している

**自アカウント実測の所見**: 伸びた投稿は共通して「1投稿1メッセージ」に絞った短尺のミニルール／気づき型で、絵文字も1つのみと簡潔。逆に伸び悩んだ投稿は『週5人に絞る』『丁寧な断り方』『写真は誰かに撮ってもらう』のように複数の主張やtipsを一文に詰め込みすぎており、情報量過多で読了・保存されにくかったと考えられる。

**改善アクション**
- [ ] トレンドの『婚活=採用面接/審査モード』のような、意外性のあるメタファー構造を自分の言葉で作り、婚活疲れ・比較疲れを言語化する短文投稿を増やす
- [ ] 1投稿1メッセージの原則を徹底し、tips羅列型（〜も、〜も）の投稿は分割して出す
- [ ] 文末を質問で締める投稿の比率を上げ、コメント誘導によるエンゲージメント率改善を狙う
- [ ] 『レッドフラッグ／グリーンフラッグ』などSNS発のトレンドワードを、断定・煽りにならない形で自分の観察に取り込みリーチ拡大を図る

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 副業

**市場トレンド（Web検索による）**
- 最初の1行で結論を書かず「気になる状態」で止める書き出し（例：断定文ではなく違和感・疑問形）が伸びやすい傾向
- 完璧に整った情報発信より、本音・弱さ・あるあるを含む「等身大」の投稿が共感を集めやすい
- 投稿末尾に問いかけを入れてリプライを誘発し、コメントで会話が生まれる投稿ほどアルゴリズム評価が上がりやすい
- リンクを含む投稿は含まない投稿に比べてリアクションが大きく落ちる傾向が報告されており、認知拡大フェーズではリンク最小化が有効
- 本文＋コメント欄に補足を続ける「二段構え」構成で滞在時間とエンゲージメントを伸ばす手法が広がっている

**自アカウント実測の所見**: 現時点でsample_size=0のため、伸びた投稿/伸びなかった投稿の実測比較はまだ行えません。まずは市場トレンドに基づく仮説（共感・問いかけ・保存要素）を含むテンプレを複数パターン投稿し、初速データ（閲覧数・返信数・保存数）を最低1〜2週間分蓄積することが優先課題です。

**改善アクション**
- [ ] 「価値提供（保存）」「共感（観察）」「質問（会話誘発）」の3パターンを最低3投稿ずつ投稿し、閲覧数・返信数を型ごとに比較できるようにログを取る
- [ ] 認知拡大フェーズ中はリンク入り投稿の比率を全体の20%以下に抑え、リンクなし投稿で信頼と会話を先に積み上げる
- [ ] 投稿冒頭1文は結論を書かず、違和感・疑問を残す書き出しに統一し、最初の2行で離脱されない構成にする
- [ ] 投稿後1時間はリプライに即応し会話を発生させることで、初速エンゲージメントを高める運用ルールを徹底する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
