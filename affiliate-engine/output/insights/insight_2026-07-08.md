# インサイトレポート 2026-07-08

## 副業

**市場トレンド（Web検索による）**
- 「副業あるある」「会社員あるある」など、特定コミュニティへの強い共感を狙った“あるある型”投稿が反応を集めやすい
- 「知らないと損する」系の損得訴求＋箇条書き情報整理タイプが保存・再投稿されやすい
- 投稿末尾に問いかけを添えてコメント欄での会話を誘発する“質問締め”がエンゲージメント向上に寄与している
- 1行目のフックを短く強くし、改行を多めにして読みやすくする文章設計が主流になっている
- 宣伝色の強い投稿・リンク付き投稿は反応が下がりやすく、価値提供とセットにするバランス感覚が求められている

**自アカウント実測の所見**: 現時点ではsample_size=0で実測データが存在しないため、伸びた/伸びなかった投稿の傾向比較はできない。まずは最低20〜30投稿分のview数・保存数・返信数を記録し、次回分析でtop3/bottom3の切り口比較ができる体制を整える必要がある。

**改善アクション**
- [ ] 投稿ごとにview数・いいね・返信数・保存数を記録し、次回分析用のデータベースを最低20投稿分蓄積する
- [ ] 「あるある共感」×「質問締め」の型を中心に週3〜5本投稿し、どの切り口が返信を集めるか比較する
- [ ] リンク付き投稿は週1本以内に抑え、必ず価値提供コンテンツとセットで出す
- [ ] 投稿時間帯を朝7〜9時・夜19〜22時に固定し、時間帯による反応差も同時に検証する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 美容

**市場トレンド（Web検索による）**
- 2026年のThreadsは単発バズよりも「なぜ起きるのか」「どう再現できるのか」を言語化した“構造のある投稿”が保存・議論・再利用されやすい傾向にある
- 美容系の反応が良い投稿は冒頭で読者の悩みを的確に突く「お悩みフック」を置き、自分ごと化させてから情報提供する型が主流
- 「乾燥肌におすすめ化粧水5選」のような情報を箇条書き・画像でまとめたランキング/比較形式は保存されやすいというデータがある
- 「美容に関する質問コーナー」のようにお題を投げかけてユーザー同士の交流を促す投稿がリプライ増加に寄与している
- 「知らないと損する」「実はやってはいけない」のようなメリット＋意外性を組み合わせた見出しが目に留まりやすいとされる

**自アカウント実測の所見**: サンプルが2件と少ないため断定はできないが、日焼け止め投稿は「パール2個分」という具体的な数値提示でいいね率は良好だった一方、ビュー数は伸び悩んだ。スキンケア投稿は「コスパ」という逆説的な切り口でビューは伸びたが、質問や行動喚起がなくいいねが極端に少なく、反応が広がりにくい構造だったと考えられる。

**改善アクション**
- [ ] 冒頭1文に「お悩みフック」を置き、読者が自分ごと化できる導入にする
- [ ] 箇条書き・比較（タイプ分け／基準リスト）で情報を構造化し、保存されやすい形にする
- [ ] 文末に具体的な二択・三択の質問を入れてリプライを誘発する（認知拡大フェーズはリンクより会話量を優先）
- [ ] リンク投稿は全体の最大1本に絞り、価値提供投稿の比率を上げてフォロー動機を強化する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 筋トレ

**市場トレンド（Web検索による）**
- Threads全体のアルゴリズムは「会話（リプライ）」を最重視する方向に進化しており、コメントが増えるほど露出が拡大する構造になっている
- 保存・議論・再利用される投稿の共通点は「なぜそれが起きるのか」「どうすれば再現できるのか」という構造化されたロジックを提示していること
- 完璧すぎる文章より、あえて「ツッコミ代（隙）」を残しリプライを誘発する投稿が評価されやすい
- 自己開示や日常の観察を含む共感型の短文投稿が保存・会話を生みやすい
- フォロワー外への配信（おすすめ欄露出）が起きやすいため、新規アカウントでも質の高い一投稿でバズが狙える

**自アカウント実測の所見**: サンプルが2件のみのため断定は難しいが、「ジムの90分よりそれ以外の23時間」という比喩・逆説型の投稿が936viewsと最も伸びており、意外性のある言い切りが閲覧数を牽引した可能性がある。一方「デスクワーカーは背中」という投稿は295viewsに留まり、いいねも0件で、具体性や問いかけ（ツッコミ代）が不足していたためリアクションに繋がらなかったと考えられる。両投稿とも「保存したくなる構造」はあるが「返信したくなる質問」が欠けている点が共通の弱点。

**改善アクション**
- [ ] 全投稿の末尾に軽い質問（例：あなたはどう？／これ意識してる？）を必ず入れ、リプライ発生率を上げる
- [ ] 逆説・比喩型の切り口（伸びた投稿の型）を継続しつつ、具体的な数字や一歩踏み込んだHow-to要素を加えて保存価値を強化する
- [ ] 投稿にあえて完璧すぎない「隙」を残し、コメントで議論・補足が生まれる余地を作る
- [ ] サンプル数が少ないため、今後2週間は同系統の切り口（逆説型・観察型）をA/Bテストし、データを蓄積してから型を絞り込む

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 教育

**市場トレンド（Web検索による）**
- Threadsのアルゴリズムは返信率を最重要指標として評価しており、問いかけで終わる投稿がリーチ拡大に直結する傾向が強い
- 育児のリアルな悩みや葛藤を率直に共有する投稿は、同じ立場の親から強い共感を集め『いいね』やコメントが伸びやすい
- 2026年のThreadsは『人間味』のある日常会話・家庭のリアルな悩みが刺さりやすく、完璧に整った情報発信よりも率直な実感ベースの投稿が広がりやすい
- 2026年度から高校無償化の所得制限撤廃や公立小学校給食費無償化など、子育て世代の家計に直結する制度変更が話題になりやすいトピックとして注目されている
- プログラミング教育・中学受験対策など『学校教育だけでは足りない部分』を扱う具体的スキル系の話題も、教育系ブログ・SNSで根強い関心を集めている

**自アカウント実測の所見**: 習い事の選定基準を『送迎の負担』という親のリアルな生活実感に落とし込んだ投稿が4040viewsと突出し、抽象的な教育論（プログラミング教育の必修化）を語った投稿は540viewsに留まった。両投稿ともview数の割にいいねが3件と少なく、共感は得られても『返信したくなる問いかけ』が弱いためエンゲージメント率（特に返信率）が伸び切っていないと考えられる。

**改善アクション**
- [ ] 『送迎の負担』のように親の生活・感情に直結する具体的な生活場面の観察を切り口の軸にし、抽象的な制度解説型の投稿は比重を下げる
- [ ] 全投稿の末尾に『あなたはどうしていますか？』等、読者が経験を投影して返信しやすい問いかけを必ず入れ、返信率を意識した3段構成（フック→価値提供→問いかけ）を徹底する
- [ ] 2026年度の高校無償化拡大や給食費無償化など、子育て世代の家計に関わる制度変更ニュースを『親目線の実感』に変換して投稿に取り入れる
- [ ] リンク入り投稿は現フェーズでは最大1本に抑え、価値提供型・問いかけ型の投稿比率を高めてフォロワー獲得を優先する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 転職

**市場トレンド（Web検索による）**
- Threadsのアルゴリズムは「いいね」より「返信率」を最重視する傾向が強まっており、返信を誘う締めの投稿設計が評価されやすい
- 「冒頭フック→具体的な価値提供→返信を誘う締め」という3段構成が伸びる投稿の型として定着しつつある
- 転職ジャンルでは「面接でのNGワード→言い換え例」のようなBefore/After型・言い換え型のリスト投稿が支持されやすい
- 夜の時間帯は一日の振り返りや共感・本音トーク系の投稿が伸びやすく、転職活動の「あるある」観察系との相性が良い
- 求人・キャリア領域では堅い告知文よりも「人柄ベース」「カジュアルな本音」の発信の方が反応を得やすい

**自アカウント実測の所見**: 現時点でサンプル数が0件のため、伸びた投稿・伸びなかった投稿の傾向を統計的に判断できる実測データがまだ揃っていません。まずは市場で確認された「返信を誘う3段構成」型の投稿を複数パターン試し、返信率・保存率を軸にデータを蓄積することが優先課題です。

**改善アクション**
- [ ] 全投稿の末尾に「あなたはどうでしたか？」型の答えやすい問いかけを必ず入れ、返信率をコアKPIとして計測を開始する
- [ ] 面接・職務経歴書・退職交渉など転職の各シーンで「言い換え・視点転換」型の価値提供リストを週1〜2本のペースで投稿し、保存率を検証する
- [ ] 夜21時前後に「転職活動あるある」の共感観察系投稿を配置し、朝・昼は情報提供系という時間帯別の投稿設計をテストする
- [ ] 認知拡大フェーズ中はリンク付き投稿を全体の1/5以下に抑え、価値提供投稿とのバランスをA/Bテストしてフォロワー転換率を確認する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## ペット

**市場トレンド（Web検索による）**
- 生成AIでの『ペット擬人化』投稿が一時トレンド1位になるなど、AI加工×ペット写真の投稿形式が话題化した
- 犬猫の『あるある』『大喜利系』のペット写真投稿がタイムラインで多く流通し、笑いや驚きを誘うテキスト・画像が拡散されやすい
- 『保護犬を迎えるべきか、ブリーダーかペットショップか』のような価値観が分かれるテーマは賛否のコメントを誘発しやすい
- Threadsアルゴリズム全体の傾向として、行動喚起（質問形式）を含む投稿はコメントが増えやすく評価が上がる
- 『知らないと損する』系の情報まとめ・チェックリスト形式の投稿は保存されやすい傾向がある

**自アカウント実測の所見**: 現状、実測データのサンプルサイズが0件のため、伸びた投稿/伸びなかった投稿の傾向を定量的に比較することができません。まずは1〜2週間、複数パターン（観察型・質問型・リスト型）を投稿してデータを蓄積することが優先課題です。

**改善アクション**
- [ ] 直近2週間は「観察+問いかけ」「保存系リスト」「賛否が分かれる中立的な質問」の3パターンを均等に投稿し、閲覧数・保存数・返信数の差を計測してデータを蓄積する
- [ ] 画像加工トレンド（擬人化・大喜利系）は模倣ではなく『構造』として参考にし、自アカウントらしい切り口（実用リスト・観察）に転用する
- [ ] コメントが荒れやすい『しつけ方針』『迎え方（保護犬/ブリーダー/ショップ）』などのテーマは断定を避け、中立的な問いかけ形式に留めてコミュニティを荒らさない設計にする
- [ ] リンク付き投稿はフェーズ方針通り最大1本に抑え、残りは保存価値・共感価値のあるテキスト投稿に集中してフォロワー純増を優先する

- アルゴリズムが「返信・会話量」を重視する方向にシフトしており、問いかけ型の投稿がリーチを伸ばしやすい
- 厚労省が2026年6月から医療機関で「睡眠障害」を診療科名として標榜可能にするなど、睡眠の悩みへの社会的関心・可視化が進んでいる
