# インサイトレポート 2026-07-14

## 転職

**市場トレンド（Web検索による）**
- Threadsは「いいね数」より「コメントでの会話量」がアルゴリズム評価に直結する会話型SNSであり、共感を呼ぶ本音投稿がコメントを集めやすい
- 「あなたは〇〇派？」形式のシンプルな質問投稿が高いエンゲージメントを生みやすい
- 自慢話ではなく反省点や工夫した点も添えた実績・体験談投稿が差別化と共感を呼びやすい
- 『乾燥肌におすすめ化粧水5選』のような箇条書き・比較形式の保存されやすい情報まとめ投稿が支持されている
- 2026年は働き方改革関連の法改正議論が進み、求職者が『法令遵守意志のある企業か』を見極める視点への関心が高まっている

**自アカウント実測の所見**: 現在アカウントの実測データはサンプル数0件のため、伸びた投稿・伸びなかった投稿の傾向分析はできません。まずは最低20〜30投稿分のビュー数・保存数・返信数を計測できる状態を作ることが優先です。

**改善アクション**
- [ ] 実測データが揃うまでは、市場トレンドで効果が確認されている『質問形式』『共感型の本音投稿』『比較・まとめ形式』の3パターンを均等に試し、初期データを収集する
- [ ] 各投稿にコメントが来たら必ず返信し、会話を継続させることでアルゴリズム評価（コメント→会話→露出増）を狙う
- [ ] 認知拡大フェーズ中はリンク投稿を全体の20%以下に抑え、価値提供型投稿を優先してフォロワー基盤を先に構築する
- [ ] 投稿ごとにビュー数・保存数・返信数を記録し、次回分析時に『伸びた投稿/伸びなかった投稿』を具体的に比較できるようにする

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## ペット

**市場トレンド（Web検索による）**
- Threadsではテキスト単体でも「共感を呼ぶ一文」が保存・会話を生みやすく、画像より言葉の投稿が伸びやすい傾向が指摘されている
- コメント・返信の多さがアルゴリズム評価に直結するため、質問形式で会話のキャッチボールを誘発する投稿が強い（インプレッションの半分は返信由来との報告も）
- 保護犬・保護猫の背景を持つストーリー性のある投稿が、単なる癒し以上の社会的共感を呼びやすい
- 猫・犬の「あるある」系の行動観察（ふてぶてしい態度、飼い主への謎の抗議行動など）がまとめサイトや大喜利的に拡散されやすい
- ChatGPT等での「ペット擬人化」画像のようなAI加工トレンドが一時的にトレンド入りするなど、季節性・話題性のある企画に便乗する動きも見られる

**自アカウント実測の所見**: 直近データはsample_size=0であり、伸びた投稿/伸びなかった投稿を比較できる実測がまだ存在しない。まずは複数投稿を継続して投稿数・閲覧数・保存/返信率のログを蓄積することが優先課題であり、現時点では市場トレンドをベースにした仮説検証型の運用に留める必要がある。

**改善アクション**
- [ ] 実測データが空のため、今後2週間は「共感観察系」「質問系」「価値提供系」の3タイプを均等投稿し、どの型が保存・返信を集めるかログを取る
- [ ] 画像なしのテキスト単体投稿と画像付き投稿を交互に試し、Threadsでの言葉主体投稿の効果を自アカウントで検証する
- [ ] 投稿末尾に必ず一言質問を添えて返信率を計測し、コメントへの返信を早めに行うことで初動エンゲージメントを高める
- [ ] リンク入り投稿は認知拡大フェーズ方針に沿って全体の3割以下に抑え、保存されやすい「お役立ちリスト」型コンテンツに限定して挿入する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 睡眠

**市場トレンド（Web検索による）**
- 「おやすみ」「もう寝ようかぁー」など、就寝前の呼びかけ・独り言系の投稿が夜間に連鎖的に拡散しやすい傾向がある
- 完璧に作り込まれた文章より、日常のふとした本音（『子どもが寝た後の静けさ』のような生活感のある一言）が保存・会話を生みやすい
- 「あなたはどう思いますか？」のような問いかけ型の一文がコメントを誘発し、アルゴリズム評価（会話量）を押し上げる
- ノウハウを箇条書き・3ステップ等に断片化した『すぐ使える情報』が引き続き好まれる形式として定着している
- 共感型ワード（わかる／それな／刺さった）で感情を刺激する一文が拡散のきっかけになりやすい

**自アカウント実測の所見**: 現状、自アカウントの投稿実績データ（サンプル数・平均閲覧・上位/下位投稿）はいずれも0件で計測されていないため、伸びた/伸びなかった要因を定量的に分析することはできません。まずは直近1〜2週間、最低でも5〜10本の投稿データを蓄積し、閲覧数・保存・返信数を記録できる状態を作ることが優先課題です。

**改善アクション**
- [ ] 投稿ごとに閲覧数・保存数・返信数を記録できるフォーマットを用意し、次回分析までに最低10投稿分のデータを貯める
- [ ] 夜間（21〜24時）と朝（7〜9時）に『共感を誘う一言＋問いかけ』を組み合わせた投稿を試し、時間帯別の反応差を比較する
- [ ] 『チェックリスト』『3つのコツ』のような断片化ノウハウ型と、『生活の一場面を描写する共感型』の2パターンを交互に投稿し、どちらが保存・返信を集めやすいか検証する
- [ ] 返信が来た投稿には必ず再返信し、会話を1往復以上続けることでスコア上昇を狙う（返信率も指標として記録する）

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
