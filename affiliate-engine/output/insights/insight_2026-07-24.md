## 婚活

**市場トレンド（Web検索による）**
- 婚活を『採用面接』や『チェックリスト審査』に例える比喩型『あるある』投稿が共感を集めやすい構造として観察できる
- 電車やカフェなど日常の中で見かけた『婚活っぽい二人』を観察者目線で描写するショートストーリー型投稿が伸びている
- 『同棲＝現状維持か結婚準備か』など男女間の意識ギャップを切り口にした心理考察系投稿が反応を得やすい
- 婚活疲れ・戦闘モードになりすぎることを諭す一般論的アドバイス投稿が一定の支持を得ている
- Threads全体の傾向として、インプレッション数よりもコメントの往復（会話設計）がアルゴリズム評価に直結するとされている

**自アカウント実測の所見**: 現時点でサンプルサイズが0件のため、伸びた投稿・伸びなかった投稿の傾向を定量的に判断できる実測データがありません。まずは市場で観察された『比喩型あるある』『観察エピソード型』『心理ギャップ型』の3パターンを試験投稿し、閲覧数・保存数・返信数を記録してベースラインを作る必要があります。

**改善アクション**
- [ ] 直近1〜2週間、比喩型・観察エピソード型・心理考察型を交互に投稿し、閲覧数/保存数/返信数を記録してどの切り口が自アカウントで機能するかを検証する
- [ ] 投稿末尾に必ず開放型の質問を設置し、コメント誘発によるアルゴリズム評価向上を狙う（Threadsはコメントのやり取りが評価されやすいとされる）
- [ ] リンク付き投稿は全体の1〜2割以下に抑え、認知拡大フェーズ中は『保存したくなる価値提供』『返信したくなる質問』を優先する
- [ ] 伸びた投稿があれば、その最初の一文（フック）の型を分析し、次の投稿の書き出しに再利用する

## 副業

**市場トレンド（Web検索による）**
- リスト形式や比較表でまとめた「あとで見返したい」投稿は保存・リポストされやすく、Threadsでは<cite index="8-1">「後で見返したい有益なリスト」や「マインドセット」などの投稿は、保存代わりにリポストされることが多い</cite>傾向がある。
- アカウントの専門性を絞り込むことが重視されており、<cite index="8-1">「美容×時短」「副業×節税」など、ターゲットの悩みを解決するテーマを1〜2つに絞り込む</cite>設計が推奨されている。
- 伸びている投稿への能動的な絡みも集客手段として使われており、<cite index="8-1">バズっている投稿の返信欄に、有益なコメントを残す</cite>ことでプロフィール流入を狙う運用が広がっている。
- 投稿頻度と型の模倣がアルゴリズム認知に直結するとされ、<cite index="9-1">毎日投稿を心がけ、積極的にバズっている投稿の型を真似るなどしてアルゴリズムに認知してもらうことが重要</cite>とされている。
- 画像・比較表を添えた「情報がまとまった」投稿は保存率が高く、<cite index="11-1">「乾燥肌におすすめ化粧水5選」のように、情報がまとまった画像は保存されやすい</cite>という構造が副業ジャンルにも応用可能。

**自アカウント実測の所見**: 自アカウントでは税金・法律系の実務ネタ（確定申告/住民税）が最も視聴数を集めており、質問形式の投稿も次点で伸びている一方、『バレる原因』のような注意喚起系は伸びが弱い。3投稿ともいいねが0件であり、視聴されても反応（いいね・保存・返信）に転換できていない点が課題。

**改善アクション**
- [ ] 確定申告・住民税など『知らないと損する実務知識』をリスト形式・箇条書きに再構成し、保存を明示的に促す一言（例：あとで見返せるように）を添える
- [ ] 質問投稿は二択形式を維持しつつ、末尾に『コメントで教えてください』のような返信ハードルを下げる一言を追加し、リプライ率を底上げする
- [ ] 『バレる』系のようなリスクトピックは、共感の観察や具体的な失敗パターンの提示を厚くして続きを読みたくなる構成に変更する
- [ ] いいね0件が続いているため、投稿末尾に軽い共感トリガー（絵文字リアクション促し）を入れ、エンゲージメント導線を明示する

## 美容

**市場トレンド（Web検索による）**
- Threadsでは冒頭1行の「悩み提示＋意外性」が拡散のトリガーになりやすい傾向
- 「乾燥肌におすすめ化粧水5選」のような情報がまとまったリスト・比較型投稿は保存されやすい
- 完璧に整った写真より、日常のリアルさや観察ベースの投稿の方が共感・コメントを集めやすい
- 売り込み色が強い投稿は警戒されやすく、役立つ情報を中心にリンク・案内はさりげなく添えるバランスが好まれる
- 「日常＋専門性」型や、決まったテーマ・タグで習慣化する「テーマ定着型」投稿が信頼形成に有効とされている

**自アカウント実測の所見**: サンプル数が1件のみのため確度は限定的だが、views1105に対しいいね2件と反応率は低め。二択で答えやすい質問締めの構造自体は市場トレンド（悩み提示＋問いかけ）に合致しているものの、保存を誘発するリスト形式や具体的な数字・比較要素が入っていない点が伸び悩みの一因と考えられる。

**改善アクション**
- [ ] 冒頭1行に「季節性の悩み＋意外性のある事実」を置くフックを強化する（例：〜だと思われがちだが実は〜）
- [ ] 箇条書き・比較形式の「まとめ投稿」を増やし、保存されやすい構造を試す
- [ ] 二択質問だけでなく、具体的なタイミングや基準を尋ねる開いた質問文でリプライの自由度を上げる
- [ ] 他ユーザーの美容関連投稿へのコメント巡回を並行して行い、露出とフォロー返しの機会を増やす

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
