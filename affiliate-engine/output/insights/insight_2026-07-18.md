# インサイトレポート 2026-07-18

## 婚活

**市場トレンド（Web検索による）**
- 婚活を「採用面接」「審査」に例え、時系列で変化する自分の思考パターンを描く自己観察型あるあるが共感を集めている
- 結婚相談所や周囲の「わきまえ論」への違和感を告発的に語る、モラハラ・クズエピソード系の観察投稿が反応を集めやすい
- 「〜すればするほどうまくいかない」型の心理法則的な言い切りに、婚活・恋愛の思い込みを重ねる投稿が伸びている
- 同棲や『一緒に住もう』の受け取り方の男女差など、価値観のズレを一言で言語化する投稿が支持されている
- 自分ではなく他人（街中で見かけた婚活っぽい2人）を観察する第三者視点の『あるある』構造も使われている

**自アカウント実測の所見**: サンプルが1件のみで統計的な結論は出しにくいが、唯一の投稿（金曜夜にアプリを見ないルール）はviews 1062に対しlike 1と、閲覧はついたもののエンゲージ（いいね・保存・返信）に繋がっていない。市場でバズっている投稿は『時系列の変化』『他者観察』『言い切りの法則化』のいずれかの構造で読者の感情や自己認識を強く揺さぶっているのに対し、自アカウントの投稿は静かな気づき止まりで、共感の余地や返信を誘う問いかけが弱いことが要因と考えられる。

**改善アクション**
- [ ] 投稿の最後に必ず1つ、読者が自分ごととして答えられる問いかけを入れて返信を誘発する
- [ ] 『はじめは〜／慣れてくると〜』のような時系列の変化構造を使い、読者に『わかる』を強く感じさせる自己観察型の投稿を増やす
- [ ] 自分の体験ではなく、婚活中の人によく見られる一般的な傾向を観察する第三者視点の投稿も試し、共感の間口を広げる
- [ ] リンク入り投稿は1本に絞り、価値提供の中に自然な形で『第三者の視点を借りる』提案として添える

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 副業

**市場トレンド（Web検索による）**
- <cite index="7-1">Threadsでは会話のキャッチボールそのものがフィードに流れ、多くの人に閲覧され、インプレッションの半分は返信から生まれるとされ、リプライは最大の集客コンテンツと位置付けられている</cite>
- <cite index="4-1">コメントから会話が生まれ、スコア上昇→露出増→バズの連鎖という仕組みがあるため、保存されやすい本音・共感系の短文投稿が伸びやすい傾向にある</cite>
- <cite index="3-7">「あなたはどう思いますか？」といった質問を投げかける行動喚起の要素がコメント増加とアルゴリズム評価向上につながるとされている</cite>
- <cite index="13-1">リンクを貼った投稿は貼らない投稿と比べてリアクションが半分以下に低下した事例が報告されており、フォロワー獲得目的の投稿ではリンク非設置が無難とされる</cite>
- <cite index="13-1">画像や動画を添付した投稿はいいねなどのリアクションが伸びる傾向が見受けられる</cite>

**自アカウント実測の所見**: 実用的な「知らないと損する」系の税務・注意喚起投稿（確定申告、バレる原因）が閲覧数トップ2を占め、情報の実利性が初速の伸びに寄与している一方、3投稿とも「いいね」が0件で、視聴はされても保存・共感・会話には至っていない。質問形式の投稿（続く人/続かない人）はビュー数こそ中間だが、会話誘発という認知拡大フェーズの目的には最も構造的に近く、今後の伸びしろが大きいと考えられる。

**改善アクション**
- [ ] 質問系投稿の比重を増やし、二択・三択形式でリプライしやすい設計にすることで会話数（＝アルゴリズム評価）を底上げする
- [ ] 投稿末尾にコメントを促す一言や絵文字を追加し、コメントには当日中に返信する運用を徹底してエンゲージメントの連鎖を作る
- [ ] 税務・バレる原因系の実用情報は「保存したくなる」フォーマット（箇条書き・チェックリスト風）に寄せて継続し、閲覧数と保存率の両立を狙う
- [ ] リンク入り投稿は認知拡大フェーズ中は月1〜2本に抑え、価値提供・質問投稿の比率を8割以上に維持する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 美容

**市場トレンド（Web検索による）**
- ハウツー・具体的手順を示す投稿が保存やフォローに直結しやすいとされ、逆に共感だけの投稿は「フォローにつながりにくい」と指摘されている
- 特定成分・アイテムの「合う肌質／合わない肌質」を切り分けて解説する投稿が2万表示超えなど大きく伸びる事例が見られる
- 冒頭1行で読者の悩みを的確に突く「お悩みフック」構成が、テキスト投稿でも重視される傾向
- 2026年美容トレンドとしてPDRNやカフェインなどの成分推し、SNSの口コミを参考にする消費行動が強まっている
- コメント返信を行うことで拡散力が上がる仕組みがあり、返信を誘発する投稿設計（二択質問・具体的な問いかけ）が有効とされる

**自アカウント実測の所見**: サンプル数2件と非常に少ないため統計的な結論は出しにくいが、いずれも「観察→軽い提案→質問締め」の同一構造で閲覧数1050〜1105とほぼ横並び、いいねも2件ずつと伸び切っていない。質問はしているものの二択・三択のような答えやすい形になっておらず、また具体的な手順（ハウツー要素）が薄いことが保存・返信の伸び悩みにつながっている可能性がある。

**改善アクション**
- [ ] 「観察+ふわっとした質問」型から、具体的な手順・基準を提示する『ハウツー型』へ比重を増やし、保存されやすい投稿を増やす
- [ ] 質問を二択・三択形式にして、読者が反射的に返信できるハードルの低い設計に変える
- [ ] 冒頭1行に「悩みフック」を置く型に統一し、最初の1秒（1行）で立ち止まらせる構成をテンプレ化する
- [ ] サンプル数が少ないため、同じ構造の投稿を最低5〜10本継続してデータを蓄積し、A/Bで『観察質問型』vs『ハウツー型』を比較する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
