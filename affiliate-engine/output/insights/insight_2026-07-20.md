# インサイトレポート 2026-07-20

## 転職

**市場トレンド（Web検索による）**
- Threads全体で「問いかけ型」投稿が反応を集めやすく、<cite index="17-4">「あなたはどう思いますか？」と問いかけ、コメントを生みやすくする</cite>構造が転職ジャンルでも「今の職場、辞める理由になりますか？」のような当事者参加型の切り口として応用されている
- 「感情フック型」「逆説型」の冒頭が支持されており、<cite index="17-4">「納得いかない」「正直しんどい」など、感情を揺さぶる一言で注意を引く</cite>手法や、<cite index="17-4">「量より質と言われるが、実は逆」など、常識をひっくり返して興味を引く</cite>逆説の切り口が転職あるあるネタと相性が良い
- 保存目的のリポストが伸びる傾向があり、<cite index="14-1">「後で見返したい有益なリスト」や「マインドセット」などの投稿は、保存代わりにリポストされることが多い</cite>ため、転職ジャンルでは「退職前チェックリスト」「面接前に見返すリスト」型が有効
- Threadsのアルゴリズムは会話量を重視しており、<cite index="17-1">Threadsはコメントのやりとりが最も重要です。いいねが多い投稿よりも、コメントでやりとりされている投稿の方が評価されやすい</cite>ため、転職の価値観を問う投稿でリプライを誘発する設計が主流
- 宣伝色の強い投稿は評価が下がりやすく、<cite index="11-2">リンクばかり貼る投稿や、自分のサービス紹介だけを繰り返す内容はスパムと見なされ、投稿の表示頻度が下げられます</cite>ため、転職エージェント紹介系もリンクは全体の一部に絞り「価値提供＋軽い誘導」の形が主流になっている

**自アカウント実測の所見**: 現状、実測データ（sample_size=0、avg_views=0、top3/bottom3なし）が取得できていないため、伸びた投稿・伸びなかった投稿の傾向比較はまだ不可能です。まずは市場で有効とされる複数フォーマット（問いかけ型・リスト型・共感観察型）を並行投稿し、初期の実測データを蓄積することを優先してください。

**改善アクション**
- [ ] 直近1〜2週間は『問いかけ型』『保存用リスト型』『共感観察型』を1日1本ずつ交互に投稿し、フォーマット別のview数・返信数を比較できるようベースラインデータを作る
- [ ] 認知拡大フェーズにつき、リンク入り投稿は全体の20%以下（5投稿に1本目安）に抑え、価値提供投稿を優先する
- [ ] 投稿後1時間は自分の投稿への返信・同ジャンル投稿への有益なコメントを積極的に行い、会話率（リプライ数）を伸ばす運用を徹底する
- [ ] 初回のデータが集まり次第、top3/bottom3の切り口・冒頭文・締め方を比較し、次サイクルのテンプレを改善する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## ペット

**市場トレンド（Web検索による）**
- AIで愛犬・愛猫を可愛くビジュアル生成する『癒し系AI動画/画像』投稿が「情報過多の現代において、多くの人は癒しを求めている」という仮説のもと大きく伸びている
- 2026年のThreadsアルゴリズムは『いいね数』より『返信数・会話の質』が最重要指標になっており、返信が自然発生する投稿ほどリーチが伸びる傾向
- 保護犬・ペットショップ論争のような賛否が分かれるテーマは強い会話を生みやすく、実際にペットジャンルアカウントでも活発なリプライ合戦が起きている
- 『ウケた型をシリーズとして横展開する』ことで一過性のバズをアカウント資産化する動きが目立つ（同じ癒し系フォーマットを犬→猫→他の動物へ展開）
- 投稿の『1行目＝フック』で興味を引き、断定を避けて『ツッコミ代（隙）』を残すことでリプライを誘発する型が2026年トレンドとして定着

**自アカウント実測の所見**: 現時点では実測データ（サンプル数0・平均閲覧数0）がなく、伸びた投稿/伸びなかった投稿の比較分析はできません。まずは投稿を蓄積し、次回分析でトップ3/ボトム3の傾向を可視化することが優先課題です。

**改善アクション**
- [ ] 認知拡大フェーズ中はまず投稿数を確保し、アルゴリズムに『アクティブなアカウント』と認識させる（データが貯まるまでは仮説ベースで複数パターンを試す）
- [ ] 断定表現を避けて『ツッコミ代』を残す一文＋末尾の質問で締める構成を標準化し、返信数を最重要KPIとして計測を開始する
- [ ] 保護犬/ペットショップのようなライトな賛否分岐テーマは、断定せず中立的な『観察・問いかけ』形式に留めてリプライを誘発する
- [ ] リンク投稿は認知拡大フェーズ方針通り最大1本に絞り、価値提供型の保存系コンテンツを軸に据える

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 睡眠

**市場トレンド（Web検索による）**
- アルゴリズムは『会話（リプライ）』を最重要視しており、コメントが続く投稿ほど拡散される傾向が強まっている
- 『本音を言うと〜』『〜ってなんか泣けるときない？』のような、一行で共感を呼ぶ短い独白型フックが保存・会話を生みやすい構造として使われている
- 投稿時間は21時〜24時台の『共感・本音トーク』枠が夜型ジャンルと親和性が高いとされる
- 質問形式の投稿（シンプルな問いかけ）はコメントを誘発しやすく、フォロワー外へのリーチ拡大に直結しやすい
- 完璧に整えた情報より、等身大でリアルな“弱さ”の吐露（オーセンティシティ）が共感エンゲージメントを生みやすい

**自アカウント実測の所見**: 直近の実測データはsample_size=0であり、投稿数・閲覧数ともに計測されていないため、現時点では自アカウントの傾向分析（伸びた投稿/伸びなかった投稿の比較）は不可能。まずは最低10〜20投稿分のデータを蓄積し、時間帯・切り口別のベースラインを作ることが優先。

**改善アクション**
- [ ] 認知拡大フェーズ中は毎日1投稿を目安に継続し、21時〜24時（就寝前の悩みが顕在化する時間帯）に固定投稿してデータを取る
- [ ] 投稿の半数以上を『質問で締める』形式にし、コメント数をKPIとして計測開始する（保存数・返信数を伸びた/伸びなかった投稿の判定基準に追加）
- [ ] 投稿後1時間はアプリを開き、来たコメントに即返信して会話を継続させる運用を徹底する
- [ ] 共感一言型・あるある型・質問型の3パターンを1週間ずつ試し、閲覧数とコメント率を比較してテンプレを絞り込む

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
