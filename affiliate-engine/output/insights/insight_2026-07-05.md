# インサイトレポート 2026-07-05

## 婚活

**市場トレンド（Web検索による）**
- 「あるある」形式で婚活初期の期待と数ヶ月後の変化（採点思考・審査モード化）をビフォーアフターで見せる構造が支持されている
- 結婚相談所やアプリで実際に言われた一言・気づきをきっかけに考えが変わる『気づき型』の投稿が反応を集めている
- 街中や日常で見かけた婚活・交際中カップルの様子を観察し、心理を読み解く『第三者観察型』の切り口が伸びている
- 「このまま一生独身かも」といった焦り・不安を素直に言語化する共感喚起型の投稿が引き続き多い
- 具体的な数値・条件（3つの条件、90分ルールなど）を提示する実用Tips型が短文でも安定して読まれている

**自アカウント実測の所見**: 上位投稿は「年齢への焦り→気づき→教訓」という感情ストーリー構造や、店選びの3条件のような実用Tipsが視聴数300超を獲得している一方、下位投稿はいずれも「プロフのリンクにまとめています」という直接的な外部誘導文言とハッシュタグ羅列で終わっており、視聴数が15〜18件と大きく沈んでいる。フォロワー数も3日間14人で横ばいのため、投稿単体のリーチはあっても保存・フォローへの転換が起きていない可能性が高い。

**改善アクション**
- [ ] 本文内で「プロフのリンクへ」と直接誘導する構成をやめ、まず本文単体で完結する価値提供型（気づき・Tips・観察）を増やす
- [ ] 感情ストーリー型（焦り→転機→教訓）を型化し、年齢・回数・エピソードの変数だけ変えて複数本量産する
- [ ] 実用Tips型（数字・条件提示）を増やし、保存されやすい「メモ化できる投稿」を強化する
- [ ] 第三者観察型（街中エピソードから婚活心理を読み解く）を新規フォーマットとして試験投稿し、反応を比較する

**新テンプレ草案**: 5本（data/proposed_templates.json に出力）

## アカウント横断アクション

- [ ] プロフへの直接誘導文言（「詳細はプロフのリンクに」等）を減らし、本文完結型の価値提供投稿の比率を7〜8割に引き上げる
- [ ] ハッシュタグを5個前後付ける投稿は伸びていないため、1〜2個の関連度の高いタグに絞る
- [ ] 感情ストーリー型・実用Tips型・第三者観察型の3フォーマットをローテーションし、反応の良い型を継続的に特定する
- [ ] リンク付き投稿は4本に1本の頻度を維持しつつ、価値提供投稿の中に自然に情報として溶け込ませる書き方をテストする
