# インサイトレポート 2026-07-17

## 転職

**市場トレンド（Web検索による）**
- Threadsのアルゴリズムは「返信率」を最重視し、次いでリポスト・いいね率の順で評価される傾向があり、返信率0.3%を超えると配信が急拡大するケースが確認されている
- 伸びる投稿の型は「冒頭フック→具体的な価値提供→返信を誘う締め」の3段構成が共通しており、最初の2行で関心を掴めないと読まれない
- 実績自慢・宣伝色の強い投稿はThreadsでは反応が悪化しやすく、フォトグラファーの実例でも実績投稿をした瞬間にいいねがつかなくなったという報告がある
- 転職市場では「AI活用を前提に業務を進められる人材」を求める企業が増え、AIをどう活用し成果につなげたかを語れることが2026年下半期の転職活動で重視されている
- 少子高齢化を背景に50代以上採用に「積極的」と答える企業が68.4%と前年比増加するなど、年齢の壁が緩む転職市場の変化が観測されている

**自アカウント実測の所見**: 現時点でsample_size=0・avg_views=0のため、自アカウントの伸びた/伸びなかった投稿の傾向はまだ分析できません。まずは直近投稿のビュー・いいね・返信数を最低20件分収集し、ベースラインを作ることが最優先です。

**改善アクション**
- [ ] 直近の投稿を「冒頭2行フック→具体的価値→返信を誘う質問」の3段構成にテンプレ化し、返信率を計測可能な状態にする
- [ ] 実績自慢・成果誇示型の投稿は避け、観察・データ紹介・問いかけ中心の投稿比率を増やす
- [ ] 投稿頻度を週5〜7本に引き上げ、反応の良い時間帯（朝・夜の共感トーク帯）に投稿を集約して比較検証する
- [ ] リンク入り投稿は認知拡大フェーズ中は最大1本/複数投稿サイクルに抑え、価値提供投稿でフォロワーとの信頼構築を優先する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## ペット

**市場トレンド（Web検索による）**
- 保護犬・保護猫の是非やペットショップ論争など、賛否が分かれるテーマがコメント数を伸ばす傾向がある
- 「ビフォーアフター」や日常のちょっとした一コマなど、リアルさのある写真・観察投稿が共感とコメントを集めやすい
- 情報がまとまったリスト・チェックリスト形式（グッズ〇選、対策まとめ等）は保存率が高くなりやすい
- 「あなたはどう思う？」型の問いかけを含む投稿はコメントが増え、アルゴリズム評価が上がりやすい
- AI生成のかわいい犬猫動画・画像やハッシュタグ企画（あるある系）を使ったアカウントも一定の反応を獲得している

**自アカウント実測の所見**: 現時点でsample_size=0のため、投稿数・閲覧数ともに実測データが蓄積されていない状態です。まずは一定期間（1〜2週間）投稿を継続し、伸びた/伸びなかった投稿を比較できるだけのデータ量を確保することが優先課題です。

**改善アクション**
- [ ] 認知拡大フェーズ中は週5〜7本を目安に投稿頻度を確保し、比較可能なデータ母数をまず作る
- [ ] 「保存されやすいチェックリスト型」と「共感を呼ぶあるある観察＋質問型」の2パターンを交互に投稿し、初期のA/Bデータを取る
- [ ] 投稿後1時間以内〜24時間以内のコメントには必ず返信し、リプライのリプライまで会話を続けて滞在時間とコメント数を伸ばす
- [ ] リンク入り投稿は全体の1〜2割以下に抑え、フェーズ方針通りフォロワー獲得を優先する構成を維持する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 睡眠

**市場トレンド（Web検索による）**
- 2026年のThreadsは会話量（返信）を軸にアルゴリズムが評価する設計に強くシフトしており、質問形式・共感一言フックの投稿が伸びやすい
- 「子どもが寝た後の静けさ」のような、生活の一場面を切り取った“共感の観察文”がそのまま保存・会話のきっかけになる型が有効
- 宣伝色の強い連続投稿はアルゴリズム上マイナス評価となるため、宣伝は1日1回以下・有益情報とセットが推奨される運用トレンド
- フォロワーが少なくても「おすすめ欄」に露出しやすい仕組みが続いており、認知拡大フェーズのアカウントには追い風の状況
- 投稿頻度は最低週7回、かつ1つの「トピックタグ」設定がリーチ最大化の鍵とされる

**自アカウント実測の所見**: 現時点でサンプル数が0件のため、伸びた投稿・伸びなかった投稿の傾向を定量的に判断できる実測データがまだ蓄積されていません。まずは投稿を継続してデータを貯め、閲覧数・保存・返信数の内訳を可視化することが最優先です。

**改善アクション**
- [ ] 直近2週間は毎日1投稿を継続し、閲覧数・いいね・返信数を記録してベースラインデータを作る（データが無い状態からの脱却が最優先）
- [ ] 1投稿の中に必ず1つ「返信したくなる質問」を入れ、コメント→会話→露出増の連鎖を狙う設計に統一する
- [ ] 宣伝（リンク付き）は1日1本以内に厳守し、価値提供投稿とセットで出す比率を高める（体感目安：宣伝1に対し価値提供4〜5）
- [ ] 睡眠ジャンル特有の「布団に入ってからの反芻思考」「寝だめできない疲れ」など生活の一場面を切り取る共感観察型を試し、どのテーマが自アカウントで反応を生むか比較する
