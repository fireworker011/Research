# インサイトレポート 2026-07-15

## 婚活

**市場トレンド（Web検索による）**
- 「婚活あるある」形式(婚活が採用面接化する等)で共感変換し、笑いと自虐を混ぜる投稿が伸びている
- 街中や日常で偶然見かけた2人の様子を観察し、婚活/相談所文化を推測する『目撃談×考察』型投稿
- 『〇〇な男性/女性の特徴7選』のようなリスト形式で保存・シェアを狙う投稿
- 潜在意識・マインド系の切り口（『負けたくない気持ちが婚活を邪魔する』等）で内省を促す投稿
- 投稿時間帯によって内容を使い分ける運用が定着しており、夜は共感・本音トーク系が特に伸びやすい

**自アカウント実測の所見**: 1投稿のみのサンプルだが、views1062に対しいいね1と反応率が非常に低く、『小さいルール』という切り口自体は市場のリスト形式・あるある形式と方向性は一致しているものの、保存動機を強める具体性（複数ルールの列挙）や返信を誘発する問いかけが本文になく、一方通行の断定文で終わっている点が伸び悩みの一因と考えられる。

**改善アクション**
- [ ] 単発の格言的な一言で終わらせず、『小さいルール◯選』のようにリスト化して保存価値を高める
- [ ] 本文の最後に『あなたはどう？』型の軽い質問を必ず添え、返信・引用を誘発する構造に統一する
- [ ] 市場で伸びている『日常観察→気づき』型や『あるある自虐』型の“構造”を取り入れつつ、自分のオリジナルの言い回し・エピソードで表現する
- [ ] 夜帯（共感・本音トーク向き）と昼帯（あるあるネタ向き）で投稿内容を使い分け、時間帯別の反応差を検証する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 副業

**市場トレンド（Web検索による）**
- Threadsは2026年にかけて『質より量』から『会話が生まれる投稿』を評価するアルゴリズムへシフトしており、<cite index="3-1">コメントが増えると会話が生まれ、そこからスコアが上昇して露出が増える仕組みになっている</cite>
- リンク投稿は反応が下がりやすいことが実務者の間でも共有されており、<cite index="1-1">リンクを貼った投稿とそうでないものを比べるとユーザーからのリアクションが半分以下に低下した事例がある</cite>
- 企業アカウントの分析でも、<cite index="6-1">インプレッションの半分は返信から生まれ、リプライは最大の集客コンテンツと位置付けられている</cite>ため、質問形式・返信を促す投稿の重要性が増している
- 2026年は単発バズよりも、<cite index="5-1">再訪率が上がる『進行中』のプロセス公開・連載型の投稿がフォロワーを引き込みやすい</cite>傾向にある
- 副業ジャンルでは『確定申告』『会社バレ対策（住民税の徴収方法・口外しない）』が定番の実務Tipsとして既に多くのメディア・アカウントで扱われており、情報自体の新規性よりも『どこで発見してもらうか』『保存したくなる形式か』が差別化ポイントになっている（<cite index="15-1">会社に副業がばれたくない場合は確定申告で住民税の納付方法を普通徴収にしておく方法が広く知られている</cite>）

**自アカウント実測の所見**: 実務Tips系（確定申告・住民税）の投稿が最も閲覧数を集めており、保存価値のある具体的な情報が視聴維持に強い一方、いいねが3投稿とも0件でエンゲージメントの『反応の薄さ』が共通課題になっている。質問投稿は2位の閲覧数だが返信を誘発する文言（リプ促進・具体的な回答フォーマット提示）が弱く、会社バレ対策投稿は市場でも既出の話題のため独自性が薄く最下位に留まったと考えられる。

**改善アクション**
- [ ] 確定申告・税務系の『保存したくなる価値提供』投稿は継続しつつ、チェックリスト形式や『◯月にやることリスト』のような構造化フォーマットに強化し保存率を上げる
- [ ] 質問投稿は二択で終わらせず、『リプで一言教えてください』など返信の心理的ハードルを下げる一文を必ず追加し、コメント起点の露出増加を狙う
- [ ] 会社バレ対策のような市場で既出の定番トピックは、体験談の捏造をせず『観察・仮説提示』の切り口に寄せて独自性を出すか、頻度を下げて別の切り口（習慣化・時間術・ツール比較など）を増やす
- [ ] 認知拡大フェーズ中はリンク投稿を全体の1/3以下に抑え、リンクを含む回でも先に価値提供を置いてから軽いCTAにする構成に統一する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 美容

**市場トレンド（Web検索による）**
- Threadsのアルゴリズムは「いいね」よりも会話量（リプライの連鎖）を重視する設計に変化しており、投稿単体の完成度より会話を生む仕掛けが評価されやすい
- 「朝と夜どっちに投稿する？」のような二択・答えやすい質問形式がコメント参加のハードルを下げ、拡散につながりやすい
- 美容系は「メリット＋意外性」の一行目（例：実はやってはいけない〇〇）で惹きつける投稿が読まれやすい傾向
- 美容師・美容発信アカウントでは、施術結果そのものより「なぜその方法を選んだか」という背景・ストーリーに共感が集まりやすい
- 2026年の美容トレンドとして韓国美容の定番化やミニマル・クリーン志向、パーソナライズ（自分の肌タイプに合わせる系）の話題が引き続き関心を集めている

**自アカウント実測の所見**: 3投稿とも閲覧数は900〜1100台とほぼ横並びだが、いいね数は「毛穴・乾燥タイプの質問」「夜スキンケアの所要時間の質問」など読者に問いかける形の投稿の方が、断定的な豆知識のみの投稿（クレンジング見直し）よりいいねが多い傾向が見える。サンプル数が3件と少なく閲覧数の差では判断しづらいが、質問形式のほうが反応（いいね・想定されるリプライ）を引き出しやすいという傾向は市場トレンドとも一致する。

**改善アクション**
- [ ] 全投稿の冒頭一文を「観察＋意外性」の型に統一し、その後に必ず答えやすい二択質問で締める構成に寄せる
- [ ] 投稿後は自分から質問へのリプライに返信し、会話を1往復以上続けることでアルゴリズム評価（会話量）を稼ぐ
- [ ] 『比較・チェックリスト型』（例：乾燥タイプ／皮脂タイプ診断リストなど）を増やし保存されやすい構造を試す
- [ ] 認知拡大フェーズ中はリンク付き投稿を全体の1〜2割以下に抑え、価値提供＋質問の投稿を主軸にする

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
