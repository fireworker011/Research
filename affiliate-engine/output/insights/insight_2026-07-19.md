# インサイトレポート 2026-07-19

## 筋トレ

**市場トレンド（Web検索による）**
- Threadsのアルゴリズムは『いいねの数よりも、どれだけリプライがつき、その会話が続いたか』を重視する傾向が強まっている（addness.co.jp）
- 一般的な常識をあえて覆す『逆説型』の書き出しが関心を引きやすく、保存や滞在時間につながりやすいとされる（akagami.blog）
- 具体的な数値を入れることで説得力が増し、保存・シェアの対象になりやすいという分析がある（akagami.blog）
- How-to系・まとめ系の投稿は保存・シェアされやすく拡散力があるが、共感系投稿は拡散向きでもフォローに直結しにくいとの指摘がある（brain-market.com）
- 会話を誘発する『質問投げかけ』や『ツッコミ代を残す』文章設計が2026年のトレンドとして重視されている（addness.co.jp）

**自アカウント実測の所見**: 自アカウントの直近2投稿は共に1000viewsを超えており露出自体は取れているが、いいねはどちらも1件のみで、閲覧に対してリアクション・会話への転換が弱い状態。特に『やめない設計』という逆説型の投稿が最も伸びており、常識の見直し系の切り口とは相性が良い一方、質問で締めた投稿（プロテインの摂取タイミング）もいいね数は伸びておらず、質問の具体性や答えやすさに改善余地がある。

**改善アクション**
- [ ] 質問で締める際は『はい/いいえ』や『一言』で即答できる具体的な質問に変え、リプライのハードルを下げる
- [ ] 逆説型フック（『〇〇と思われがちだが実は〜』）を継続しつつ、数値（週◯回、◯g、◯ヶ月など）を必ず1つ入れて保存動機を強化する
- [ ] 投稿末尾に『保存してあとで見返してね』等の軽い行動喚起を添え、保存率を可視化・向上させる
- [ ] リンク投稿は最大1本に抑え、価値提供型2本を軸にして認知拡大フェーズの目的に沿ったバランスを維持する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 教育

**市場トレンド（Web検索による）**
- Threadsは2026年にかけて「リプライ数・会話の連鎖」が拡散の最重要シグナルとなっており、質問形式で返信を誘発する投稿が伸びやすい
- 抽象的な精神論よりも「制度変更・費用」など生活に直結する具体的な時事ネタ（給食費無償化など）が保存・会話を生みやすい
- 「なぜそうなるのか」を一言で言語化する構造化された投稿（比較・二択形式）が保存されやすい傾向
- 完璧に整った文章より、本音や等身大の言葉・生活のリアルな一場面を切り取った投稿の方が共感を呼びやすい
- 画像より言葉（テキスト単体）の方がThreadsでは伸びやすく、冒頭2行で続きを読ませる工夫が重要

**自アカウント実測の所見**: 伸びた3投稿はいずれも「送迎負担」「宿題の家庭比較」「給食費無償化」など生活に直結する具体的なテーマで、質問や二択で締めて読者の参加を促していた。一方伸びなかった3投稿は「体験教室の見方」「プログラミング適性」「叱り方の設計」など一般論的なアドバイス調で終わり、問いかけが弱く読者が反応する余地が少なかった。

**改善アクション**
- [ ] 全投稿の末尾に具体的な二択・質問を必ず入れ、リプライ（会話）を誘発する構造に統一する
- [ ] 時事性のある制度・費用系トピック（給食費、教材費、学童保育料等の変化）を定期的に取り入れ、保存・議論を狙う
- [ ] 抽象的な教育論を投稿する際も『あなたの家庭は？』『どちらを選びますか？』のような二択・比較構造に変換し、読者が自分ごと化できるようにする
- [ ] リンク投稿は認知拡大フェーズ方針通り最大1本に絞り、価値提供投稿の比率を高める

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 節約

**市場トレンド（Web検索による）**
- Threads全体で「問いかけ型」の投稿（例：あなたはどう思いますか、という参加を誘う一言）がコメントを集めやすく、アルゴリズム評価にもつながりやすいとされている
- 感情フック型・逆説型・権威借り型など、常識をひっくり返す切り口や共感を呼ぶ一言が伸びる型として整理されている
- Threadsのアルゴリズムは「いいね」よりも返信・会話の継続を重視する傾向があるとされ、投稿後にリプライが連鎖するかどうかが拡散の鍵とされている
- 画像より言葉（テキスト）そのものが伸びやすく、日常の本音や気づきを綴った短文が保存・会話を生みやすいという分析がある
- 過度な宣伝やリンクの多用はスパム的と判断されやすく、宣伝は1日1回以下に抑え役立つ情報とセットにするのが望ましいとされている

**自アカウント実測の所見**: 現時点で自アカウントの投稿実績データ（サンプル数・平均表示・上位/下位投稿）がゼロのため、伸びた/伸びなかった要因を定量的に特定できる段階にありません。まずは一定期間・一定投稿数のベースラインデータを蓄積し、次回以降で型ごとの比較検証を行う必要があります。

**改善アクション**
- [ ] 認知拡大フェーズ中は、価値提供型・共感観察型・質問型の3パターンを均等にローテーションして投稿し、初期データを蓄積する
- [ ] 投稿頻度を一定に保ち、新規アカウントとしての活動量をアルゴリズムに認知させる（毎日でなくても短文投稿を継続する）
- [ ] 投稿後はできるだけ早い時間帯にリプライへ返信し、会話の連鎖を意識的に作る
- [ ] リンク付き投稿は全体の中で最大1本に抑え、価値提供とセットにすることで宣伝色を薄める

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
