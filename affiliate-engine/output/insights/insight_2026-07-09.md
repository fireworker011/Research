# インサイトレポート 2026-07-09

## 婚活

**市場トレンド（Web検索による）**
- 婚活を「採用面接」「審査」など別の構造にたとえて笑いと共感を同時に取る“再定義型あるある”が伸びている
- 感情フック（納得いかない・しんどい）で入り、最後に価値観の転換を一文で落とす構成がバズ投稿に多い
- Threadsは『いいね数』より『コメントのやりとり』がアルゴリズム評価に直結するとされ、答えやすい質問で締める投稿が伸びやすい
- 3行程度の短文＋絵文字1つで、最初の一行に意外性やメリットを置く投稿がインプレッションを稼ぎやすい
- 個人の武勇伝より『〜という声が多い』『〜と気づいた人がいる』という伝聞・観察のトーンが共感を集めやすい

**自アカウント実測の所見**: 自アカウントで最も伸びた投稿（610・426views）は「〇〇の正体は〜」型の“気づき・再定義”構造を持ち、下位3本（314・301・282views）は実用Tipsのみで気づきの一文や質問フックが弱い。いいねが最も付いた「34歳で入籍」投稿は個人の実体験を断定的に語る構造で市場トレンドと一致するが、社内ルール上この形式（具体的な偽の体験談化）は今後避け、観察・伝聞ベースに寄せる必要がある。

**改善アクション**
- [ ] 冒頭一文を「〇〇の正体は△△」「〇〇な人は、実は〜」のような再定義・逆説フックに統一し、閲覧維持率を上げる
- [ ] 全投稿の末尾に、YES/NOや一言で答えられる具体的な質問を必ず設置し、コメント誘発によるエンゲージ連鎖を狙う
- [ ] 体験談は一切禁止のため『〜という声が多い』『〜と気づいた人がいる』という観察・伝聞形で共感を作る
- [ ] 認知拡大フェーズはリンクを最小限（1投稿のみ）にし、価値提供投稿の比率を高めて保存・返信を優先する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 副業

- いいね数よりも「リプライがつき会話が続くか」がアルゴリズム評価の中心となっており、問いかけで返信を誘発する設計が重視されている

- 『専門性のない雑記アカウント』はパーソナライズ機能により埋もれやすくなっており、ジャンルを一貫させた発信が中長期的なフォロワー資産化に有効とされている

**改善アクション**
- [ ] まずは投稿ごとに『閲覧数・いいね・リプライ数・保存数』を記録し、次回分析用にサンプルを蓄積する

## 美容

- How-to系（役立つ具体的Tips）投稿は保存・シェアされやすくフォロワー獲得に直結しやすい一方、共感系はバズりやすいがフォロー率は低いという傾向が指摘されている
- 文末に「あなたはどう思いますか？」等の質問を入れて行動喚起することで返信が増え、アルゴリズム評価が上がるという構造が有効とされている
- 「実はやってはいけない〇〇」「知らないと損」といった意外性フックが最初の一文で興味を引く型として使われている
- 美容・ダイエットジャンルは「肌荒れが治った方法」など、すぐに試したくなる具体的な情報が保存・拡散されやすいジャンル特性を持つ
- 完璧に作り込んだ内容より人間味のあるリアルな声のほうが共感を呼びやすく、過度な演出より素の観察・気づきが響きやすい

**改善アクション**

- [ ] 全ての投稿の文末に「あなたは何秒派？」等の具体的な質問を追加し、返信率を意図的に引き上げる
- [ ] 日焼け止め投稿のようにいいね率が高かった『具体的数値×行動指針』型のHow-to投稿を軸テンプレとして強化する
- [ ] 共感・あるある型（例：ケアの失敗あるある）を1〜2本試し、views/いいね/返信の反応差を比較データとして蓄積する

- リスト化・チェックリスト形式やHow-to系投稿は『また見返したい』と思わせやすく、保存されやすい型として支持されている
- 『あなたはどう思う？』のような答えやすい質問を投げかける対話型の投稿が、コメント増加とアルゴリズム評価向上につながっている
- 完璧に作り込んだ内容より、生活密着型で人間味のある“等身大”の観察・気づき投稿が共感を呼びやすい
- 教育・子育てジャンルでは、送迎や家庭学習の負担など『親のリアルな悩み』に切り込む切り口が伸びやすい

## 睡眠

- 『眠れない夜に誰もが考えること』のような、あるある共感型のリスト形式コンテンツは睡眠ジャンルで根強く支持されている
- 特定の体勢や動作を使った具体的でニッチな入眠テクニックを丁寧に解説する投稿は、保存されやすいコンテンツとして機能している
- ハッシュタグではなく1つの『トピックタグ』設定と質問形式の投稿が、会話誘発とリーチ拡大の鍵になっている
