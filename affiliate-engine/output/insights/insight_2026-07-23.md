# インサイトレポート 2026-07-23

## 転職

**市場トレンド（Web検索による）**
- Threadsのアルゴリズムは「返信率」を最重視し、次いでリポスト・引用、いいね率の順で評価される傾向があるため、転職ジャンルでも問いかけ型の投稿が伸びやすい
- 冒頭2行で興味を引く「フック→具体的価値提供→返信を誘う締め」の3段構成が高エンゲージ投稿の共通パターンとして観察されている
- 「知らないと損する」「たった一つ」「初心者でも」といった具体性・意外性のある言い回しが反応を得やすい傾向がある
- 教育系・意見系・問いかけ系・事例系など複数タイプの投稿を組み合わせることで、単一ジャンル固定よりも幅広い層にリーチできる
- 朝は前向き・応援系、夜は共感・本音トーク系など時間帯によって刺さる切り口が変わる傾向があり、転職の悩み投稿は夜間との相性が良いと考えられる

**自アカウント実測の所見**: 現時点で投稿実績データ（サンプル数0）がなく、伸びた投稿・伸びなかった投稿の傾向を定量的に比較できない状態です。まずは複数タイプ（あるある共感／質問形式／価値提供リスト）の投稿を試し、返信数・保存数を軸に初期データを蓄積することが優先課題です。

**改善アクション**
- [ ] 投稿タイプを「共感あるある」「保存用リスト」「質問投げかけ」の3系統に分けて最低5〜10投稿ずつ試し、返信率・保存率で初期ベンチマークを作る
- [ ] 冒頭1〜2行に具体性（数字・意外性ワード）を入れるフックを全投稿で統一し、離脱を防ぐ
- [ ] リンク誘導は認知拡大フェーズの方針通り最大1本/複数投稿に留め、残りは価値提供・共感投稿で信頼構築を優先する
- [ ] 投稿時間帯を朝（前向き系）・夜（共感・本音系）で分けてテストし、フォロワー層の反応が良い時間帯を特定する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## ペット

**市場トレンド（Web検索による）**
- 複雑な意味づけをしない『純粋な癒し』系コンテンツ（可愛い動物の姿をそのまま見せる）が強く反応を得やすい
- 投稿末尾での質問・呼びかけ（行動喚起）がコメント増加とアルゴリズム評価向上に直結している
- 完璧に作り込んだ内容より、生活感のある等身大の投稿の方が共感を呼びやすい傾向
- 保護犬の迎え方・しつけ方針など『意見が分かれるテーマ』は賛否のコメントを呼び拡散されやすい
- ペットの『あるある行動』や仕草の観察投稿は保存・共有されやすいジャンルとして定着している

**自アカウント実測の所見**: 現時点では投稿実績データ（sample_size=0）が存在せず、伸びた投稿・伸びなかった投稿の傾向を分析できる材料がありません。まずは複数パターンの投稿を試行し、インサイト機能で表示回数・保存数・コメント率を記録してベースラインを作ることが優先課題です。

**改善アクション**
- [ ] 直近1〜2週間、切り口の異なる投稿（観察系・質問系・あるある系）を最低5〜7本ずつ投稿し、インサイトで反応差を記録してベースラインデータを作る
- [ ] 全投稿の冒頭1行に『スクロールを止める共感フック』を入れる型を統一し、A/Bテストする
- [ ] 投稿末尾に質問形式のCTAを標準搭載し、コメント率・保存率の変化を測定する
- [ ] 認知拡大フェーズ中はリンク付き投稿を最大1/5本程度に抑え、価値提供投稿を優先してフォロー転換率を検証する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）

## 睡眠

**市場トレンド（Web検索による）**
- テキスト中心で「本音・共感トーク」を投げかける投稿が伸びやすい傾向。<cite index="1-1">Threadsは画像より言葉のほうが圧倒的に伸びやすい</cite>
- コメントを誘発する一言投稿が拡散の起点になっている。<cite index="1-1">コメント → 会話 → スコア上昇 → 露出増 → バズの連鎖 という仕組みがあるから（hottolink）</cite>
- 「保存版リスト」や「後で見返したいtips」形式がリポスト（保存代わり）されやすい。<cite index="11-1">保存代わりのリポスト「後で見返したい有益なリスト」や「マインドセット」などの投稿は、保存代わりにリポストされることが多い</cite>
- 夜21〜24時台は「共感・本音トーク」系が特に反応しやすい時間帯とされる。<cite index="9-1">朝は「今日も頑張ろう系」、昼は「息抜きになる話題」、夜は「共感や本音トーク」が伸びやすいです</cite>
- 同ジャンルの伸びている投稿の返信欄に価値あるコメントを残す「出向き型」の露出獲得も有効視されている。<cite index="11-1">同ジャンルで既に多くのフォロワーを抱えるアカウントや、バズっている投稿の返信欄に、有益なコメントを残しましょう</cite>

**自アカウント実測の所見**: 実測データが0件（sample_size=0）のため、伸びた投稿・伸びなかった投稿の傾向をまだ分析できない状態。まずは1〜2週間、異なる切り口・時間帯で投稿し、閲覧数・保存/返信の初動データを蓄積することが優先課題。

**改善アクション**
- [ ] データ収集を最優先し、1週間で最低5〜7投稿を「共感観察型」「リスト型」「質問型」に分けて投下し、閲覧数・返信数を記録する
- [ ] 投稿時間を夜21〜24時台に寄せてテスト運用し、朝・昼投稿との反応差を比較する
- [ ] 睡眠ジャンルで伸びている他アカウントの投稿にコメントで絡み、プロフィール流入経路を作る
- [ ] 保存されやすい「工夫リスト」型フォーマットを週1回は必ず投下し、保存/リポスト率の変化を観察する

**新テンプレ草案**: 3本（data/proposed_templates.json に出力）
