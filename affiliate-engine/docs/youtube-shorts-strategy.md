# YouTube Shorts 第三弾 — 戦略とジャンル選定（2026-07 リサーチ版）

第1弾（ペット）・第2弾（婚活）に続く第三弾チャンネルの立ち上げ方針。
市場リサーチ〜投稿ネタ出しは `src/shorts-research.js` が毎朝自動で行い、
採用・動画化・アップロードの判断だけを人間が行う（半自動）。

## 運用フロー（半自動の全体像）

```
毎朝 自動（GitHub Actions affiliate_engine_shorts.yml）
  shorts-research.js
    ├ チャンネル別に Web検索で市場リサーチ（伸びている傾向・フック・型）
    ├ 同ジャンル Threads アカウントの実測を補助シグナルとして突き合わせ
    ├ 台本+Grok画像プロンプトのネタを生成（タイトル/フック/ナレーション/
    │  画像プロンプト4〜6枚+動画化指示/タグ/CTA/リンク）
    ├ compliance.js で誇大表現・薬機法リスクをブロック
    └ Issue「🎬 YouTube Shorts ネタ出しキット」に配信 ＋ 採用待ちJSONを保存
        ↓
人間（採用ネタを選ぶ）
  node src/shorts-production-kit.js --idea <ID>
    → output/shorts/production/<ID>/ に以下を生成
      kit.md                手順+CapCutチェックリスト+概要欄メタ（リンクURL入り）
      grok_agent_prompt.txt Grok Agent Mode 用の一括生成プロンプト
      narration.txt         Grok Companion 読み上げ用の台本
        ↓
人間（制作・30〜60分）
  1. grok_agent_prompt.txt を Grok Agent Mode に貼る → 画像生成〜動画化まで一括実行
  2. narration.txt を Grok Companion に読ませて音声を抽出
  3. CapCut でクリップ+ナレーション+BGM+字幕を編集（チェックリスト参照）
        ↓
  YouTube Studio で予約投稿（概要欄に link_key のアフィリエイトリンク+#PR）
```

## 案件差し替え運用（商品非依存の原則）

「良い案件が見つかったら差し替える」を前提に、**商品とコンテンツを分離**してある。
商品特化の紹介動画は案件終了と同時に資産価値を失うが、悩み解決型のジャンル動画は
リンクを貼り替えるだけで生き続ける（YouTube 攻略の定石としても、単なる商品紹介より
How To・比較などの悩み解決型が推奨されている）。

- **仕組み**: `config/links.json` の各キーは「案件スロット」。ネタ・台本・CTA は
  商品名/ブランド名/価格を含まない設計（shorts-research.js のプロンプトで強制）で、
  CTA は「概要欄にまとめてあります」型の汎用誘導のみ
- **第三弾の主力案件**: `第三弾_主力案件` キーに現在の A8 案件 URL を登録済み。
  差し替え時はこの URL を書き換えるだけ。動画・台本・スケジュールは触らない
- **過去動画**: 案件差し替え後、概要欄のリンクを新 URL に貼り替える（YouTube は
  投稿後も概要欄を編集できるため、過去動画も新案件の導線として再稼働する）
- **投稿前チェック**: kit.md に「リンクが生きているか投稿前に開いて確認」を組み込み済み
  （案件終了・在庫切れは収益ゼロに直結する定番事故）

## 第三弾ジャンル: 美容（確定 2026-07-10）

主力案件（オルビスユー ドット、A8 提携済み）の取得に伴い、第三弾は美容ジャンルで確定。
案件スペック: **新規購入 1,800円 / EPC 830 / 確定率 97.8%**（EPC・確定率ともに優良。
確定率 97.8% は非承認リスクがほぼ無いことを意味し、収益予測が立てやすい）。

### この選択の根拠（2026-07 時点のリサーチ）

1. **ショート動画が美容コンテンツの主戦場**: 2026年は TikTok / YouTube Shorts / リールが美容の主戦場で、30秒完結のテクニック系・習慣系がバズりやすいとされる
2. **ターゲットと案件が一致**: オルビスユー ドットの想定層（30〜45歳女性のエイジングケア入門）は、購買意欲の高い層に届きやすいと言われる年代向け美容発信とそのまま重なる
3. **Threads 美容アカウントの実測が使える**: 既存の Threads 美容アカウント（ゆるエイジングケア路線）の top_posts 実測を補助シグナルに流用できる（threads_genre: 美容）。切り口の勝ちパターンが Shorts に横展開できる
4. **既存2チャンネルと客層が重ならない**: ペット（癒し）・婚活（恋愛）・美容（Health/Beauty）で HARM ジャンルの分散が効く

※ 以前推薦していた「節約・お金」のリサーチは第四弾候補としてそのまま有効
（格安SIM・ふるさと納税のリンクスロットも links.json に維持済み）。

### チャンネル設計

- **コンセプト**: 「がんばらない30代からのエイジングケア」。ズボラ×時短の習慣・雑学を1本1トピック。Threads 美容アカウントと同じ世界観（成分解説の専門系は競合が強いため避ける）
- **フォーマット**: Grok 生成のシネマティックな素材（スキンケア静物・手元・ルーティン風景。顔出しなし・文字焼き込みなし）+ Grok Companion ナレーション + CapCut テロップ
- **名前の方向性**: 「◯◯｜がんばらない大人のスキンケア」「夜10分のゆるケア手帖」系
- **投稿ペース**: 開設2週間は1本/日でアルゴリズムに学習させ、以降は反応を見て調整
- **導線**: 価値提供3〜4 : リンク誘導1。誘導は「ケアの見直し」「夜の習慣」文脈で概要欄へ（商品名は動画に出さない＝案件差し替え耐性）
- **⚠️ 薬機法（美容は表現規制が最重要）**: 「シミが消える」「若返る」「肌質が変わる」等の効能断定は NG。「うるおい」「ハリ感のある印象」「年齢に応じたケア」の範囲内で書く。config/youtube.json の `caution` に明文化してあり、ネタ生成プロンプトに毎回自動注入される。「(肌|顔|見た目)が若返る」等は compliance.js でもブロック

### 立ち上げ手順（残作業）

1. YouTube チャンネルを開設し、`config/youtube.json` の beauty チャンネルに `handle` を記入、`status` を `active` に変更
2. デモとして `output/shorts/production/shorts_beauty_20260710_1`（リンク誘導型）と
   `shorts_beauty_20260710_2`（価値提供型）の制作キットを生成済み。まずこの2本で
   Grok Agent Mode → Grok Companion → CapCut のフローを一周し、所要時間と仕上がりを確認する
3. 翌朝から毎朝のネタ出しキットに美容チャンネル分が届く（candidate の間も「開設検討中」印付き）

## KPI と撤退ライン

- Shorts の収益はアフィリエイト（概要欄リンク）が主。再生数は手段であって目的ではない
- 開設30日: 平均再生が2チャンネル既存実績の半分未満 → フック・型を全面見直し（ネタ出しキットの winning_formats を再確認）
- 開設60日: リンククリックがほぼゼロ → 案件かCTA文脈のミスマッチを疑い、link_key の差し替えを検討
