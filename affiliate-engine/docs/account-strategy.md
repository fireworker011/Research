# アカウント戦略 — 4スロット再設計（2026-07 市場リサーチ版）

前提: A8.net で提携済みの案件と現行スロット（転職・VOD）が**ミスマッチ**（両ジャンルとも案件ゼロ）。
提携済み案件から逆算してスロットを再設計する。

## 0. 最優先: 既存婚活アカウントの即収益化

新アカウントを作る前に、**稼働中の婚活アカウントに提携済みリンクを設定する**のが最短の収益化。

| 案件 | 成果地点 | 備考 |
|---|---|---|
| ウェルスマ（オンライン仲人型相談所） | 無料相談等 | リーズナブル訴求が Threads の「アプリ疲れ」文脈と好相性 |
| パートナーエージェント | 資料請求・無料カウンセリング | 結婚相談所は資料請求のみで数千円の成果地点があり、新規入会だと相場4万円級の高単価ジャンル |

**穴場ポジション: 「アプリ疲れ → 相談所」の橋渡し。**
マッチングアプリ訴求のアカウントは飽和しているが、「アプリで消耗した30代が相談所を検討する」文脈の発信は少ない。
既存テンプレ（アプリの立ち回り）で共感を取り、相談所比較へ誘導する導線は競合が薄い。

→ やること: `config/links.json` の「婚活」にウェルスマ or パートナーエージェントのリンクを設定。
テンプレは次回再生成時に「相談所検討」文脈を追加。

## 1. Threads で伸びるアカウントの共通項（2026年時点の公開情報より）

- **会話が起きる投稿が最優先評価**。投稿後の初動（数十分）のいいね・返信が拡散を決める。「答えたくなる質問」で締めるのが最も簡単で効果的
- 伸びやすい時間帯は **7〜9時 / 12〜14時 / 21〜24時**（現行スケジュール 7/12/19/21時は概ね適合。19時→21時台重視は検討余地）
- フォロワーが少なくても伸びる設計のため**新規参入向き**
- 収益化アカウントの定石は **有益投稿3〜5 : リンク投稿1**（現行テンプレ比率 96:84 はリンク比率が高め。再生成時に 4:1 へ寄せる）
- 成約率が高いのは HARM ジャンル（Health健康・美容 / Ambition キャリア・副業 / Relation 恋愛・婚活 / Money お金）

## 2. スロット別の再設計

### スロット1（旧: 転職）→ **お金・副業**

- **使える案件**: ココナラ（購入者募集）、A8メディア会員募集、mixhost、楽天、**DMM FX（口座開設系で1〜5万円級の最高単価。ただし成果条件は口座開設+5万円入金+取引1回と重い）**
- **穴場**: 「誰でも月○万」系の誇大アカウントが飽和 → **逆張りの「盛らない副業記録」**。現実的な数字（月1〜3万円）を淡々と公開する信頼型は希少で、コメントが付きやすい
- **ターゲット**: 30〜40代の会社員・主婦（収入の柱を増やしたい層）
- **名前案**: 「ゆう｜盛らない副業記録」「さき｜手取り21万の家計と副業」「けい｜会社員のまま月+3万」
- **アイコン**: 実写風 or シンプルな人物イラスト。ネイビー×白で「誠実・堅実」。札束・高級時計系の画像は信頼を落とすので厳禁
- **投稿の型**:
  - 収支公開型: 「今月の副業収支を公開。売上◯円、かかった時間◯h。時給換算すると…」→ 質問締め
  - 失敗談型: 「最初にやって損した副業3つ」
  - 手順型: 「ココナラで最初の1件が売れるまでにやったこと」
- **導線設計**: ココナラ・A8登録・mixhost（ブログ開設）は行動ハードルが低く初成果向き。DMM FX は「余剰資金でやる」文脈でリスク明記の教育型投稿のみ
- **⚠️ 法規制**: 金融は誇大表現の規制が最も厳しい。利益の断定・保証は不可、FX は必ずリスク併記。compliance.js の既存ブロックに加え、投資助言に当たる個別売買推奨は書かない

### スロット2（美容）→ 継続。ただし軸を「30代からのゆるエイジングケア」に絞る

- **使える案件**: オルビスユー ドット（エイジングケア）、リフリーラ
- **穴場**: 成分解説系（○○酸がどう…）は専門アカウントが強い → **「ズボラ×時短ケア」**。「疲れて何もしたくない夜に最低限これだけ」という文脈は共感が起きやすく、30代後半〜40代のオルビスユー ドットのターゲットと一致
- **ターゲット**: 35〜50歳女性、仕事・育児で時間がない層
- **名前案**: 「みほ｜がんばらない35歳のケア」「りさ｜夜10分だけのスキンケア」
- **アイコン**: パステル・ベージュ系の柔らかい色調。コスメのフラットレイ写真 or 柔らかい人物イラスト
- **投稿の型**:
  - 最低限型: 「時間がない夜、削っていいケアと削っちゃダメなケア」
  - 共感型: 「30代半ばで肌の『戻る力』が変わった気がする。同じ人いる？」→ 会話誘発
- **⚠️ 薬機法**: 効果断定（シミが消える等）は compliance.js がブロック。「使用感」「個人の感想」の枠で書く

### スロット3（旧: VOD）→ **メンズボディメイク（忙しい30代の時短筋トレ×食事）**

- **使える案件**: HMBマッスルプレス、Muscle Deli（宅食）、リフリーラ
- **リサーチ所見**: 一般男性の筋トレアカウントは伸びにくい（マッチョ実績者が強い）→ **特化型が必須**
- **穴場**: 「デスクワーカーの週2×30分」「体を変えるのはジムより食事」という**時短・食事寄り**ポジション。Muscle Deli（食事）と HMB（サプリ）の2案件に導線が自然につながる
- **ターゲット**: 30〜45歳のデスクワーク男性（腹が出てきた・時間がない）
- **名前案**: 「たく｜週2ジムの30代」「しん｜デスクワーカーの体づくり」
- **アイコン**: ジムでの後ろ姿・シルエット系（顔出し不要で「実践者感」が出る）。黒×オレンジ等
- **投稿の型**:
  - ビフォー過程型: 「3ヶ月続けて分かった、やらなくていい筋トレ」
  - 食事型: 「体は台所で作られる。コンビニで選ぶならこの3つ」
  - 質問締め: 「みんな週何回ジム行けてる？」
- **⚠️ 薬機法**: サプリで「筋肉がつく」と断定しない。「トレーニングのお供」の位置づけ

### スロット4（旧: 複合）→ **子育て×教育（習い事・プログラミング教育）**

- **使える案件**: ヒューマンアカデミージュニア（ロボット・プログラミング教室、全国展開）、楽天、Furbo
- **リサーチ所見**: 教育系は Threads で会話が起きやすく、中学受験ジャンルで発信開始2ヶ月でフォロワー1万人・月15万円に到達した事例が公開されている。子育て層は Threads のボリュームゾーン
- **穴場**: 「プログラミング教育必修化・大学入試の情報Ⅰに焦っている親」。危機感はあるが何をすべきか分からない層が厚く、「教室に通わせる前に家で試せること」→「教室の選び方・体験会」という導線が自然
- **ターゲット**: 小学生の子を持つ30〜45歳の親
- **名前案**: 「はな｜小学生ママの習い事メモ」「ゆき｜プログラミング教育、正直どうなの」
- **アイコン**: 親子の柔らかいイラスト。黄色・水色系の明るい配色
- **投稿の型**:
  - 費用リアル型: 「習い事の月謝、全部書き出したら◯円だった。みんなどうしてる？」→ 最強の会話誘発
  - 検討記録型: 「プログラミング教室の体験に行って分かった、見るべきポイント3つ」
  - 楽天で知育玩具・学習グッズの紹介も混ぜられる（複合性を維持）

## 3. アイコン共通ガイドライン

- 1080×1080、**顔（or キャラクター）が円形トリミングでも判別できる大きさ**
- ジャンルカラーで背景統一（副業=ネイビー / 美容=ベージュ / 筋トレ=黒 / 教育=黄・水色）
- 5アカウントで**同じ作風のイラストは使い回さない**（同一運営者と機械的に紐づく見え方は避け、各アカウントを独立したメディアとして育てる）
- 実在人物の写真・他アカウントの画像の流用は不可。AI生成イラスト or 自分の写真素材

## 4. 「オマージュ」の運用ルール

バズアカウントの**構造（フック→本文→質問締め）・投稿時間・有益:リンク比率**を学ぶのは戦略。
**文面のコピー・言い回しの丸写し・プロフィール模倣は盗用**であり、Threads の重複コンテンツ判定でリーチも落ちる。
本システムでは `output/top_posts.json`（自アカウントの実測勝ちパターン）を few-shot にする設計になっており、
これが最も安全で再現性のある「型の学習」。他アカウントから取り入れるのは「型の分類」（上記の◯◯型）まで。

## 5. 立ち上げ順序（推奨）

一気に4アカウント開設はしない（初動の会話対応が追いつかず全部中途半端になる＋短期間の複数アカウント一斉開設はスパム挙動に見えやすい）。

1. **今週**: 婚活アカに相談所リンク設定（既存資産で即収益化）
2. **Week 2**: 「子育て×教育」開設（会話が起きやすく、事例的に Threads との相性が最も良い）
3. **Week 3-4**: 「お金・副業」開設（DMM FX の単価が目標達成のレバレッジ最大）
4. **Week 5 以降**: 婚活・教育・副業の実測を見て、筋トレ or 美容軸変更を判断

各アカウント開設時: `config/accounts.json` の genre を更新 → Secrets 登録 → `--from-file` でスケジュール再生成。
