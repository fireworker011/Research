# SNS運用 15系統システム

3 SNS × 5ジャンル = **15系統**。各系統は専用フォルダの `SYSTEM.md` を持つ。
**1系統 = 1会話 = 1md** で運用する。共通ルールは [`CLAUDE.md`](./CLAUDE.md)。

## 使い方(スマホ)
1. 新しい会話を作る
2. 対象系統の `SYSTEM.md` の「起動コマンド」1行を貼る
3. 作業 → 終了時にClaudeが知見を `SYSTEM.md` に追記してcommit
