## 2. 最も効果的だった投稿タイプと理由

### 🥇 第1位：trust_building（信頼構築型）
**平均視聴数：89.7view**（全タイプ中ダントツ1位）

特に投稿③「パラダイムシフト」の**179view**が牽引。

#### なぜtrustbuildingが強いのか

**Threadsの拡散メカニズムと合致しているから。**

Threadsは「知らない人の投稿でも共感・知見があれば表示する」レコメンドフィードが強い。trust_buildingタイプは以下の要素を自然に含みやすい。

- **具体的な数字**（20社・100人・8ヶ月などミキのペルソナはこれが最強の武器）
- **読者の代わりに失敗してくれた経験談** → 「これ私のことだ」という共感
- **情報密度が高い** → 最後まで読まれやすい

#### 投稿③が突出した理由（推測）
「パラダイムシフト」というワードは**認知の転換を予感させる**。婚活で悩んでいるユーザーが「え、何が変わるの？」と思ってタップした可能性が高い。**好奇心ギャップ（情報格差）の設計が偶発的にハマった**と考えられる。

### 🥈 第2位：paradox（逆説型）
**平均視聴数：22.0view**

サンプル数1本のため断言はできないが、逆説・意外性は**スクロールを止める力**が強い。今後強化すべきタイプ。

## 3. 改善が必要なタイプと具体的改善案

### 🔴 最重要改善：affiliate（アフィリエイト投稿）
**平均視聴数：5view　← アカウント最低値**

#### 問題の本質
アフィリエイト投稿は**フォロワー100人未満では機能しない**。

現在の状態で「パートナーエージェントに申し込んでください」と訴求しても：
- 見る人がいない
- 信頼残高がゼロ
- 「売り込み」に見えてむしろ逃げられる

#### 具体的改善案
```
【今すぐやめること】
→ 直接的な申込み訴求投稿

【代わりにやること】
→ 「なぜ私が結婚相談所を選んだか」という
  個人的な意思決定ストーリーに変換する

【投稿設計の転換】
Before：「パートナーエージェントがおすすめです！→URL」
After ：「アプリで100人に振られた私が、最終的に
        結婚相談所に辿り着いた理由【3つの転換点】
        （詳細はプロフのリンクから）」
```

---

### 🟡 要改善：fomo（緊急性訴求型）
**平均視聴数：5view**

「今すぐやらないと損」という訴求は、**信頼関係が構築されていない相手には不快感を与えるだけ**。初期フェーズでのFOMO投稿は逆効果。フォロワー500人超えるまで封印推奨。

---

### 🟡 要改善：advice（アドバイス型）
**平均視聴数：12.8view**

数は多いが視聴数が伸びない。原因は**「どこにでもある婚活アドバイス」になっている可能性**。

#### 改善の方向性

```
【弱いアドバイス投稿のパターン】
「プロフィール写真は自然な笑顔が大切です」
「まずは相手の話をよく聞きましょう」

【ミキが書くべきアドバイス投稿のパターン】
「100人と会った私が気づいた、
 2回目デートに繋がる男性の"たった1つの共通点"」
 
→ 数字 × 自分の実体験 × 意外性 の組み合わせ
```

---

### 🟡 要改善：personalstory（個人ストーリー型）
**平均視聴数：14.0view**

ストーリーの**具体性が足りない**と推測。

```
【改善前イメージ】
「昔、婚活がうまくいかなくて辛かった時期がありました」

【改善後イメージ】  
「28歳の夜、マッチングアプリの通知が
 また『いいね0人』だった。
 お風呂の中で泣いたのは、あの夜で何回目だったか。
 ────
 あれから6年後の今、私は入籍届を出した。」
```

**場面・感情・セリフ・数字**を入れることで映像として浮かぶ文章に変換する。

## 4. 投稿時間帯の評価と推奨

### 現状評価
データ上、12:00・21:00ともに**投稿実績ゼロ**。

これは**最も重要な時間帯に投稿されていない**ことを意味する。

### Threadsにおける婚活ペルソナの推奨時間帯

ターゲット（30代・婚活中の女性）の行動パターンから導出：

| 時間帯 | 推奨度 | 理由 |
|--------|--------|------|
| **12:00〜12:30** | ⭐⭐⭐⭐⭐ | 昼休みのスマホタイム。コンテンツ消費のピーク |
| **21:00〜22:00** | ⭐⭐⭐⭐⭐ | 帰宅後のリラックスタイム。感情移入しやすく「保存・シェア」が増える |
| **07:00〜08:00** | ⭐⭐⭐ | 通勤中。短文投稿との相性は良い |
| **23:00〜** | ⭐⭐ | 夜更かし層に届くが、婚活層はこの時間帯に疲弊していることが多い |

### 📋 推奨投稿スケジュール（週7本目安）

```
月：21:00（confession/告白系で週の始まりに共感を集める）
火：12:00（advice/短くて役立つTips）
水：21:00（personal_story/感情的なストーリー）
木：12:00（paradox/逆説で昼の暇つぶし層を引き込む）
金：21:00（trust_building/週末婚活を控えた読者に刺さる）
土：12:00（trust_building/週末アクティブユーザーが多い）
日：21:00（personal_story or confession/内省タイムに響く）
```
