## 2. 最も効果的だった投稿タイプと理由

### 🥇 第1位：**confession（告白・本音型）**
> avg_views: 16.0 / avg_ER: **1.93%**（唯一ERを記録）

```
confession投稿（views:26, likes:1, ER:3.85%）
→ 全14投稿中、唯一エンゲージメントが発生
```

**なぜ効くのか：**

Threadsユーザーの行動心理として、**「自分だけが感じていると思っていた感情を言語化してくれた投稿」** にのみ反応が起きます。告白・本音型は「あ、私も同じだった」という**共鳴スイッチ**を押す力が最も強い。

ミキさんのペルソナ（6年間・100人超で全滅した経験）は、**confession型との相性が構造的に最高**です。失敗の数・深さ・感情的リアリティが、他の婚活アカウントには絶対に出せない差別化になります。

### 🥈 第2位：**trust_building（信頼構築型）**
> avg_views: **32.3**（全タイプ最高）

投稿⑤「プロフアドバイス」の63PVは全投稿トップ。ただしいいね0。
→ **「読まれてはいるが、反応する理由がない」** 状態。有益情報は届いているが、感情的フックが欠如。

### 🥉 第3位：**paradox（逆説型）**
> avg_views: 22.0 / 1本のみ

サンプル数が少ないものの、ビュー数はconfessionに次ぐ水準。「婚活で頑張れば頑張るほど遠ざかる」系の逆説は、**Threadsの拡散ロジックに乗りやすいテーマ**。

## 3. 改善が必要なタイプと具体的改善案

### ❌ 最優先改善：**affiliate（アフィリエイト型）**
> views: **5**（全タイプ最低）/ ER: 0%

**問題の本質：**
フォロワー・信頼残高がゼロの段階での直接的な商業訴求は、SNSでは「見知らぬ人からの営業電話」と同等の拒絶感を生みます。

**改善アクションプラン：**

```
❌ 現状：「パートナーエージェントはおすすめです→詳しくはこちら」

✅ 改善後の設計図：

【ステップ1】confession投稿でミキの婚活失敗体験を深く語る
     ↓
【ステップ2】「なぜアプリ婚活が私に合わなかったか」を分析するtrust_building
     ↓
【ステップ3】「結婚相談所に偏見があった私が変わった理由」でパーソナルストーリー
     ↓
【ステップ4】自然な文脈でアフィリエイトに誘導（比較検討型）
```

**アフィリエイト投稿の書き換え例：**
> 「28歳から6年、マッチングアプリ20社で全滅した私が、最終的に選んだのは結婚相談所でした。当時の私なら絶対に信じなかった選択。でも、あの決断がなければ今の旦那さんと出会えていない。興味ある人だけ見てね→[リンク]」

---

### ⚠️ 要改善：**advice（アドバイス型）**
> avg_views: 12.8 / ER: 0%（4本すべて反応なし）

**問題：** 情報が「正しいが、冷たい」状態。婚活メディアの記事と差別化できていない可能性が高い。

**改善案：**
```
❌ 「プロフィール写真は明るい笑顔が大切です」
　　（情報だけ・感情ゼロ）

✅ 「プロフィール写真、私も最初は完全に間違えてた。
　　スタジオで撮った完璧な写真を使ってたのに、
　　マッチング率ほぼゼロ。気づいたのは3年後。
　　理由は衝撃的で...【続きを読む的な構成】」
　　（自分の失敗体験＋情報の組み合わせ）
```

---

### ⚠️ 要改善：**fomo（希少性・緊急性型）**
> views: 5 / ER: 0%

**問題：** フォロワーベースがない段階でのFOMO訴求は機能しません。「誰も知らない人が限定と言っても...」という心理的反応が起きる。

**改善案：** 現時点ではFOMO投稿は**完全停止を推奨**。フォロワー500人を超えてから再導入を検討。

## 4. 投稿時間帯の評価と推奨

### 現状評価

```json
"12:00": {"count": 0, "avg_er": 0.0}
"21:00": {"count": 0, "avg_er": 0.0}
```

**致命的問題：** 推奨時間帯（12時・21時）への**投稿実績がゼロ**です。時間帯比較が機能していない状態。

### 📅 推奨投稿スケジュール

```
🎯 最優先時間帯
├── 21:00〜21:30（婚活ターゲット層がスマホを手にする時間）
│   理由：30代女性の就寝前スクロール時間。感情的コンテンツが響きやすい
│
├── 12:00〜12:15（昼休み）
│   理由：短時間で消費できる投稿との相性が高い
│
└── 07:30〜08:00（朝の通勤時間）
    理由：「今日も婚活頑張ろう」の感情と重なるタイミング

⚠️ 避けるべき時間帯
├── 15:00〜17:00（エンゲージメント率が業界全体で低い）
└── 23:00以降（翌日のアルゴリズム評価に乗りにくい）
```

**来週の実験計画：**
- confession型 → **21:00投稿**
- trust_building型 → **12:00投稿**
- personal_story型 → **07:30投稿**
→ 3パターンを1週間試してデータ収集
