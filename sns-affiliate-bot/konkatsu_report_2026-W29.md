## 2. 最も効果的だった投稿タイプと理由

### 🏆 trust_building（信頼構築型）が唯一の機能タイプ

**上位3投稿の共通パターン分析：**

```
1位「年齢のせいじゃなかった」    → 402views / ER 0.75%
2位「婚活やめたいと思った日」    → 335views / ER 0.60%
3位「夜中に泣いた話」           →  93views / ER 1.08%
```

**なぜこのタイプが機能するか：**

**① タイトルに「否定→反転」構造がある**
「年齢のせいじゃなかった」は婚活女性が最も恐れている「年齢コンプレックス」を正面から取り上げ、かつ「じゃなかった」で希望を示している。**読者の心理的ペインと出口を1行で表現**できており、クリック必然性が高い。

**② 感情的ピークが明確**
「婚活やめたいと思った日」「夜中に泣いた話」は、**誰もが心の中で思っているが言語化できていない感情**をそのままタイトルにしている。Threadsユーザーは「わかる」と感じた瞬間に立ち止まる。

**③ ミキのペルソナが活きるテーマ**
6年・100人超という圧倒的な経験値は、成功体験より**失敗・消耗・限界**を語るときに最も説得力を持つ。このタイプはペルソナとの親和性が最も高い。

## 3. 改善が必要なタイプと具体的改善案

### ❌ advice型：ビュー15・ER 0.0%（致命的不振）

**根本原因の診断：**

advice型が失敗する理由は**「誰が言っているかより、何を言っているか」を優先した設計**になっているため。現段階のフォロワー数・知名度では、アドバイスを受け取る信頼残高がまだ不足している。

**具体的改善策：**

| 現状の問題 | 改善アクション |
|-----------|--------------|
| 「専門家的アドバイス」口調 | 「私がやって変わった〇〇」の体験談形式に変換 |
| タイトルなし（空白） | 必ず数字か感情ワードを入れたタイトルを設定 |
| trust建設前のadvice投入 | trust_building：advice＝8：2の比率に固定 |
| 抽象的な内容 | 「パートナーエージェントに行った初日」など具体固有名詞を使用 |

**advice型の正しい使い方（当面の方針）：**
> 純粋なアドバイスは封印し、**「私が体験して気づいたこと」フォーマット**に統一する。結果的にアドバイスになるが、入口は必ず自分の体験談にする。

---

### ⚠️ 低ビュー投稿の共通問題点

```
「プロフ修正しても変わらなかった」 →  9views
「メッセージで消耗した2年間」      →  5views
「自分を偽って疲れた」             → 13views
```

**共通する敗因：**

1. **タイトルが「過程」で止まっている**——「変わらなかった」「消耗した」で終わると、読者に得られるものが見えない
2. **Threadsの初期表示（冒頭2行）で興味を引けていない可能性が高い**
3. **投稿時間が最適化されていない**（次項参照）

**改善フォーミュラ：**
```
❌ 「メッセージで消耗した2年間」
⭕ 「2年間メッセージだけで終わった私が、やめた理由」
            ↑ペイン提示          ↑アクション＋続きへの引き
```

## 4. 投稿時間帯の評価と推奨

### 現状：12:00・21:00ともに投稿実績ゼロ

データ上、時間帯別の最適化が**まったく行われていない**状態。これは機会損失として最も即効性のある改善ポイント。

**Threadsアルゴリズムと婚活ターゲット（30代女性）の行動パターンを踏まえた推奨：**

| 時間帯 | 推奨度 | 理由 |
|--------|--------|------|
| **12:00〜12:30** | ⭐⭐⭐⭐⭐ | ランチ休憩中のスマホ閲覧ピーク。30代会社員女性が最も婚活に思いを馳せる時間帯 |
| **21:30〜22:30** | ⭐⭐⭐⭐⭐ | 帰宅後・入浴後のリラックス時間。感情移入しやすく、共感→保存行動が起きやすい |
| **7:00〜8:00** | ⭐⭐⭐ | 通勤電車。流し読みされやすいが、刺さればER高い |
| 14:00〜16:00 | ⭐ | 業務中のため到達率低い |

**来週からの投稿スケジュール（固定ルール化推奨）：**
```
月・水・金 → 12:00投稿
火・木・土 → 21:30投稿
日曜      → 9:00投稿（週末モーニングルーティン層を狙う）
```

## 5. 次週の投稿戦略

### 基本方針：「共感の量」から「共感の深さ」へ

現状はビューは取れているが**リプライゼロ**という状態が継続している。アフィリエイト転換には「この人に聞いてみたい」という心理が必要で、そのためには**読者が返信したくなる「問いかけ」「余白」「続きへの期待」**を投稿設計に組み込む必要がある。

### 優先タイプ比率

```
trust_building（体験談×反転構造）  ： 70%（5本）
advice（体験談フォーマットに変換）  ： 0%（封印）
新規：question型（問いかけ終わり）  ： 30%（2本）
```

### 重点テーマ設定

| 優先度 | テーマ | 狙い |
|--------|--------|------|
| 🔴 最優先 | 「戦略変更の転換点」 | 何が変わったかへの好奇心を喚起→アフィリエイトへの自然な接続 |
| 🔴 最優先 | 「婚活相談所vs.アプリの本音比較」 | パートナーエージェントへの関心誘導 |
| 🟡 重要 | 「100人会ったから言える〇〇」 | 実績の権威性強化 |
| 🟡 重要 | 読者への問いかけ投稿 | リプライ数改善・コミュニティ形成 |
