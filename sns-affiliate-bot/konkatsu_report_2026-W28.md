## 3. 改善が必要なタイプと具体的改善案

### ⚠️ 課題①：advice型の完全不発（15views / ER 0.0%）

**原因分析：**
タイトル未設定（空白）のまま投稿されており、**フックが存在しない状態**。Threadsでは最初の1〜2行でスクロールを止められなければ即スルーされる。

**改善案：**
```
❌ 現状：タイトルなし・内容不明
✅ 改善：「28歳の私に教えたかった、婚活で最初にやるべきたった1つのこと」
         → 数字・対象・ベネフィットを冒頭に凝縮
```

### ⚠️ 課題②：低視聴投稿の共通パターン

```
「プロフ修正しても変わらなかった」  →  9 views
「メッセージで消耗した2年間」        →  5 views
「自分を偽って疲れた」               → 13 views
```

**原因：**
- **「失敗・消耗・疲れ」だけで終わっており、読者に次の行動や希望を与えていない**
- タイトルが「暗さの描写」で止まっており、転換点（=読む理由）がない

**改善フレームワーク：**
```
❌ 「疲れた・消耗した・変わらなかった」（暗さで終わる）
✅ 「疲れた→でもそれが転機だった」（暗さ＋光の予感）
✅ 「変わらなかった→変えるべきはここだった」（問題提起＋答えの予告）
```

### ⚠️ 課題③：replies（返信）が全投稿で0件

**最重要改善項目。**Threadsのアルゴリズムはreplies数を強く評価する。

**改善案：投稿末尾に「問いかけ」を必ず入れる**
```
例：「婚活中に一番しんどかった瞬間って、どんな時でしたか？
     コメントで教えてもらえると嬉しいです🙏」
```
