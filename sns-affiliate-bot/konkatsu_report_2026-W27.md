## 2. 最も効果的だった投稿タイプと理由

### 🏆 今週のトップ3投稿

```
1位「年齢のせいじゃなかった」    views:401 / ER:0.75%
2位「婚活やめたいと思った日」    views:334 / ER:0.60%
3位「友達に打ち明けた日」        views: 69 / ER:1.45%（ER最高）
```

### 共通する成功要因の分析

**① タイトルに「反転・否定構造」がある**
> 「年齢のせいじゃなかった」→ 多くの婚活女性が抱く"年齢への諦め"を真正面から否定。読者が「私もそう思ってた」と感情移入しやすい。

**② 普遍的な挫折感に触れている**
> 「婚活やめたいと思った日」→ 婚活経験者なら一度は感じる感情。共感の間口が広い。

**③ 「友達に打ち明けた」は人間関係という第三者視点が入り、ストーリーに奥行きが出た**

### 📌 なぜviews低投稿が生まれたか

```
「プロフ修正しても変わらなかった」→ views:8
「メッセージで消耗した2年間」    → views:4
```

これらは**「手法の失敗談」**であり、読者の心理的フックが弱い。婚活Threadsでは「感情」が刺さるが、「作業の失敗」は刺さらない。

## 3. 改善が必要なタイプと具体的改善案

### 🔴 問題①：投稿タイプが「trust_building」一択

**現状の致命的リスク：**
Threadsのアルゴリズムは多様なエンゲージメント（保存・引用・リプライ）を重視します。信頼構築系の感情投稿だけでは「共感されても動かない」状態が続きます。

**改善案：投稿タイプの3本柱を導入する**

```
【推奨配分（週9本の場合）】

Trust Building（感情共感）  ▓▓▓░░░░░░ 3本（現在9本→削減）
教育・情報提供              ▓▓▓░░░░░░ 3本（現在0本→新規追加）
行動喚起・CTA              ▓▓▓░░░░░░ 3本（現在0本→新規追加）
```

### 🔴 問題②：リプライがゼロ＝会話が生まれていない

**原因：** 投稿が「独白」で終わっており、読者への問いかけがない

**改善案：「返報性の問いかけ」を毎投稿末尾に追加**

```
❌ 現在（独白型）
「6年間でたくさんのものを失った気がした。」

⭕ 改善後（問いかけ型）
「6年間でたくさんのものを失った気がした。
あなたは婚活で"これが一番きつかった"って
思う瞬間、ありましたか？」
```

### 🔴 問題③：フォロワー獲得への導線がない

全投稿にプロフィールへの誘導文・フォロー促進の一言が確認されません。

**改善案：投稿末尾テンプレートの固定化**

```
---
📌 6年・100人超の婚活を経て8ヶ月で入籍したミキが
  同じ悩みを持つ方に向けて毎日発信しています
  → フォローで続きが読めます
```

## 4. 投稿時間帯の評価と推奨

### 現状評価：🔴 致命的な問題あり

```json
"12:00": { "count": 0, "avg_er": 0.0 }
"21:00": { "count": 0, "avg_er": 0.0 }
```

**12時・21時への投稿が1本もない状態**です。これはThreadsで最もリーチが取れるゴールデンタイムを完全に捨てている状況を意味します。

### 推奨投稿時間帯（婚活女性ターゲット・根拠付き）

```
🥇 21:00〜22:00（最優先）
   理由：帰宅後のリラックスタイム。感情系コンテンツへの
         共感感度が最も高まる時間帯。婚活アプリも
         この時間帯が最もアクティブユーザー数が多い。

🥈 12:00〜12:30（次点）
   理由：昼休みのスマホ閲覧ピーク。
         短く刺さる投稿との相性が良い。

🥉 07:00〜07:30（補助枠）
   理由：通勤時間帯。「今日も頑張ろう」という
         モチベーション系投稿と相性が良い。
```

## 5. 次週の投稿戦略（優先タイプ・テーマ）

### 戦略の核心：「共感で集め、情報で信頼し、CTAで動かす」

```
PHASE 1：感情で引き込む（月・木・日）
  ↓
PHASE 2：情報で「この人は信頼できる」と思わせる（火・金）
  ↓
PHASE 3：具体的な行動（相談・クリック）に誘導する（水・土）
```

### 優先テーマ設定

**感情共感テーマ（今週の成功パターンを踏襲）**
- 「反転・気づき」構造を維持
- 末尾に必ず問いかけを入れる
- 例：「〇〇だと思ってたけど、違った」形式

**教育・情報テーマ（新規追加）**
- 「婚活アプリ20社使った私が唯一続けられた理由」
- 「プロ婚活と自己流の決定的な違い3つ」
- ※パートナーエージェントへの自然な橋渡しになるテーマを選ぶ

**行動喚起テーマ（新規追加・アフィリエイト導線）**
- 直接的なCTAは避け、「背中を押す体験談」形式で
- 「無料相談に行く前に私が怖かったこと、正直に話します」
