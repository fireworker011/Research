# YouTube Shorts アフィリエイト自動化システム 仕様書

## システム概要

ペット×見守りカメラをテーマにしたYouTube Shortsアフィリエイト動画を、
画像生成からSNS投稿まで半自動で制作・運用するシステム。

---

## アーキテクチャ全体像

```
[台本生成] → [画像生成] → [動画化] → [音声生成] → [動画合成] → [投稿] → [分析・リサーチ]
  手動/AI      自動化済     手動        自動化済     自動化済      自動化済    自動化済
```

---

## 機能一覧

### 1. 台本生成（手動 / AI支援）

**概要**
7シーン構成のショートドラマ台本を生成する。各シーンに以下を含む：
- ナレーションテキスト（Voicevox / Grok Companion用）
- 画像生成プロンプト（gpt-image-1用）
- テロップテキスト（ASS字幕用）

**台本構成（7シーン）**
| scene | 役割 | 内容 |
|-------|------|------|
| 1 | フック | 視聴者の手を止める感情的な一文 |
| 2〜3 | 状況説明 | 背景・感情の文脈を作る |
| 4〜5 | 展開 | ペットカメラ映像・発見・感情 |
| 6 | 転換/オチ | 感動・笑い・驚きの山場 |
| 7 | CTA | 商品訴求＋「コメント欄のリンクへ」 |

**フックパターン（実績あり）**
- 感情直球型：「泣いてたの、バレてた」「心配で、泣いてしまった」
- 状況描写型：「帰れない夜だった」「玄関で、目が合った」
- 不安×数字型：「一人で8時間、何してるんだろう」
- 共感型：「仕事中、何度スマホを見たかわからない」
- 笑い×謎型：「カメラに映ってたの、見せていいですか」「真顔で待ち伏せされてた」
- 常識破壊型：「ペットカメラって、カメラじゃなかった」

**手を止めるフック30選（5パターン）**

常識破壊：
1. ペットカメラって、カメラじゃなかった
2. 留守番が得意な猫なんて、いない
3. 一人でいるほうが楽、は人間の思い込みだった
4. 帰ってから謝っても、もう遅い
5. 10年飼って、初めて知った
6. うちの子のこと、わかってると思ってた

共感：
7. 仕事中、何度スマホを見たかわからない
8. 帰り道、いつも走りたくなる
9. 朝、行ってきますって言えなかった日がある
10. ご飯をあげるだけでいい、って思ってた時期があった
11. 泣いてたの、気づかれてた
12. 名前を呼んだら、来てくれると思ってた

不安：
13. 今日、ちゃんとご飯食べたかな
14. 一人で8時間、何してるんだろう
15. もし倒れていたら、気づけない
16. うちの子、もしかして泣いてる？
17. 出張3日目、限界だった
18. 最後に顔を見たのは、何日前だっけ

憧れ：
19. 会社からでも、声をかけられる時代になった
20. 帰る前に、ただいまって言える
21. 名前を呼んだら、走ってきてくれた
22. 声だけで、しっぽが動いた
23. 離れていても、目が合う
24. 世界中どこにいても、うちの子の顔が見られる

数字訴求：

26. 留守番中、猫の行動の7割は飼い主が知らない

28. ペットカメラを買った人の9割が「もっと早く買えばよかった」と言う
29. 声かけ機能を使った飼い主の8割が、猫の反応が変わったと回答
30. ペット見守りカメラ、去年から利用者3倍

### 2. 画像生成（自動化済）

**スクリプト**: `generate_images_petX.py`

**使用モデル**: gpt-image-1

**パラメータ**
```python
model   = "gpt-image-1"
size    = "1024x1536"   # 縦型2:3
quality = "high"
format  = "b64_json"    # URL形式ではなくbase64
```

**キャラクター設定（女性の一貫性を保つ共通プロンプト）**
```
Japanese woman in her early 30s, black shoulder-length straight hair
slightly tucked behind one ear, oval face, natural makeup,
wearing a light beige turtleneck sweater and dark navy trousers
```

**共通スタイル指定**
```
Photorealistic, cinematic, Japanese setting, no text or watermarks.
```

**場面別プロンプト設計のルール**
- 女性が登場するシーン：WOMAN変数を先頭に追加して一貫性を確保
- ペットカメラ映像シーン：「Pet camera footage style, slightly grainy security camera look.」を追加
- 縦型構図：全シーンに「Vertical 9:16 portrait.」を追加
- 首切れ防止：「Full upper body in frame, face and hair visible.」を明示

**選択的再生成**
```bash
python generate_images_petX.py 2 3 5   # 指定シーンのみ再生成
```

### 3. 動画化（手動 / Grok Agent）

**ツール**: Grok Agent（xAI）

**手順**
1. scene1.png〜scene7.pngをGrok Agentに一括アップロード
2. 以下の形式でプロンプトを投入：

```
以下の画像7枚から、それぞれ5秒の縦型動画（9:16）を生成してください。
すべて同時に生成をお願いします。

【scene1.png】
（シーンごとの動きの指示）

共通設定：
- 各動画5秒
- 縦型 9:16
- テキスト・ウォーターマークなし
- シネマティックな自然な動き
```

**注意**: Grok動画生成APIは現時点で非公開。Agentモードのみ利用可能。

### 4. 音声生成

#### 4a. Voicevox（自動化済）

**スクリプト**: `make_audio_petX.py`

**パラメータ**
```python
SPEAKER_ID = 9       # 波音リツ
SPEED      = 0.88    # 感動系
PITCH      = -0.03
INTONATION = 1.4

SPEED      = 0.90    # 笑い系
INTONATION = 1.3
```

**エンドポイント**
```
POST http://localhost:50021/audio_query?text={text}&speaker={id}
POST http://localhost:50021/synthesis?speaker={id}
```

#### 4b. Grok Companion（手動）

感情を込めた音読。ナレーション全文を投入し、音声入り動画をダウンロードして使用。
APIは非公開のため手動のみ。

### 5. 動画合成（自動化済）

#### 5a. Voicevox音声を使う場合
**スクリプト**: `assemble_petX.py`

#### 5b. Grok Companion音声を使う場合（CapCut不要）
**スクリプト**: `assemble_grok.py`

**処理フロー**
```
Grok Companion動画
  ↓ FFmpegで音声抽出（CapCut不要）
  ↓ 無音区間を自動検出して7分割
  ↓ 各シーン動画を音声尺に自動フィット
  ↓ キャプション（ASS字幕）自動生成・焼き込み
  ↓ BGM自動ミックス（音量比 BGM:ナレ = 0.18:1.0）
  ↓ 縦型1080×1920でエクスポート
完成動画
```

**使用方法**
```bash
python assemble_grok.py               # 無音検出で自動分割
python assemble_grok.py --no-split    # 均等分割（BGM混入時）
```

**キャプション設定（ASS形式）**
```
フォント: Meiryo 55px
色: 白文字・黒縁（Outline 4px）
位置: 下部中央
```

### 6. YouTube投稿（自動化済）

**スクリプト**: `modules/publishing/youtube_publisher.py`

**認証**: OAuth2.0（token.pickle にキャッシュ）

**投稿設定**
```python
title       = "動画タイトル"
description = "説明文\n\n#ハッシュタグ"
tags        = ["猫", "ペットカメラ", "見守りカメラ"]
category    = "22"   # People & Blogs
privacy     = "public"
```

**Shorts化の条件**
- 動画の縦横比：9:16（1080×1920）
- 尺：60秒以内
- タイトルまたは説明に `#Shorts` を含める

**ピン固定コメントテンプレート**
```
▼ 動画で紹介したペットカメラはこちら👇
https://px.a8.net/svt/ejp?a8mat=4B5UWA+GC8AGI+3SUY+5Z6WY

✅ 留守番中に声をかけられる
✅ スマホで24時間確認できる
✅ 夜間撮影・双方向通話対応

気になる方はお気軽にコメントください🐾
#PR #ペットカメラ #見守りカメラ
```

**重要**: Shortsの説明欄リンクは2023年8月以降タップ不可。コメント欄ピン固定が必須。

### 7. 競合リサーチ・チャンネル監視（自動化済）

**スクリプト**: `monitor_and_research.py`

**機能**
1. 自チャンネル全動画の再生数・いいね・コメントを取得
2. 競合キーワードでYouTube検索（再生数順）
3. 結果をMarkdownレポートとして保存
4. オマージュ候補TOP5をコンソール出力

**検索キーワード**
```python
COMPETITOR_KEYWORDS = [
    "ペットカメラ ショートドラマ",
    "保護猫 感動 shorts",
    "見守りカメラ 猫 shorts",
    "ペット 留守番 感動",
```

## 運用ルール

1. **投稿間隔**: 2〜3日（毎日投稿はアルゴリズムに不利）
2. **投稿後**: 必ずコメント欄にアフィリンクをピン固定
3. **リサーチ**: 投稿のたびに `monitor_and_research.py` を実行
4. **タイトル**: 短く・謎・感情・問いかけのどれかを含める
5. **視聴率目標**: 80%以上（Shortsアルゴリズムの基準）
