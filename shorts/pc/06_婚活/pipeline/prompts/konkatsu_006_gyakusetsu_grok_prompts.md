# konkatsu_006_gyakusetsu — Grokアプリ用プロンプト一覧(ステップ実行版)

各シーンごとに、①画像プロンプトをGrokに貼って画像生成 →
②できた画像を選んで動画化ボタン →③動きの指示(動画プロンプト)を貼る。
完成したらファイル名を `scene_01.mp4`〜`scene_06.mp4` にして
このチャットに送る(またはinput/フォルダに配置)。

**注意**: ナレーション音声は後からパイプライン側で合成します。
動画生成時は「口を閉じたまま、話しているように見える動きはさせない」よう
動画プロンプトに必ず含めてください(リップシンクのズレ防止)。

---

## シーン1 (目安10秒)

**① 画像プロンプト(コピペ用)**
```
30代前半の日本人女性、茶色がかった黒髪ミディアム、細面、清楚感のある顔立ち。ベージュのニット。縦長9:16のリアルな写真風。 自宅のソファでカメラの方を見つめる女性。口は閉じたまま、話していない。少し考え込むような真剣な表情。柔らかい室内照明。 テキストなし。
```

**② 動画化の動きの指示(コピペ用)**
```
口を閉じたまま、話さない。カメラがゆっくり近づく。女性の表情にフォーカス。まばたきと微かな表情の変化のみ。
```

保存ファイル名: `scene_01.mp4`
