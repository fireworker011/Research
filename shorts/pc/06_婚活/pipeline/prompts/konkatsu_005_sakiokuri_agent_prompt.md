# Grok Imagine Agent Mode — 一括実行プロンプト(konkatsu_005_sakiokuri)

> Grok Imagine の Agent Mode にこのブロックをそのまま貼ってください。
> 画像生成→動画化を6シーン分、Agentが自動で連続実行します。
> テロップ・ナレーション音声・BGMは不要(パイプライン側で自動合成するため)。

---

あなたは婚活ジャンルのショート動画クリエイターです。
以下の6シーンについて、①画像生成 → ②その画像を動画化、を順番に自動で行ってください。
音声・BGM・テキスト(テロップ)は一切不要です。各シーンの動画だけ作ってください。

【重要】ナレーション音声は別撮りで後からパイプライン側が合成します。
動画内の人物は口を閉じたまま、話しているように見える動きは一切させないでください
(リップシンクさせると、後から乗せるナレーションと口の動きがズレて不自然になります)。
表情の変化・まばたき・うなずき・視線の動きだけで演技してください。

【全シーン共通のキャラクター設定・必ず統一すること】
30代前半の日本人女性、茶色がかった黒髪ミディアム、細面、清楚感のある顔立ち。ベージュのニット。縦長9:16のリアルな写真風。
全シーンで同一人物として描くこと。

---

### シーン1(保存名: scene_01.mp4 / 尺目安10秒)
**画像**: 自宅のソファでカメラの方を見つめる女性。口は閉じたまま、話していない。少し後悔のにじむ真剣な表情。柔らかい室内照明。 テキストなし。
**動画化**: 口を閉じたまま、話さない。カメラがゆっくり近づく。女性の表情にフォーカス。まばたきと微かな表情の変化のみ。

### シーン2(保存名: scene_02.mp4 / 尺目安10秒)
**画像**: 窓際で頬杖をつき、遠くを見つめる女性。口は閉じたまま、話していない。壁にカレンダー。夕方の光。物思いにふける表情。 テキストなし。
**動画化**: 口を閉じたまま、話さない。窓からの光がゆっくり変化する。カメラ固定。

### シーン3(保存名: scene_03.mp4 / 尺目安10秒)
**画像**: 夜、ベッドでスマートフォンの課金画面を見る女性。口は閉じたまま、話していない。スマホの光が顔を照らす。 テキストなし。
**動画化**: 口を閉じたまま、セリフを話さない。カメラ固定。女性が小さく息を吐き、スマホを置く(発話なし)。

### シーン4(保存名: scene_04.mp4 / 尺目安10秒)
**画像**: 結婚式場の前を一人で歩く女性。口は閉じたまま、話していない。幸せそうなカップルの背中が遠くにぼけて見える。少し切ない表情。夕暮れ。 テキストなし。
**動画化**: 口を閉じたまま、話さない。カメラがゆっくり引く。女性が立ち止まって遠くを見る。

### シーン5(保存名: scene_05.mp4 / 尺目安10秒)
**画像**: 明るいカウンセリングルームでスーツ姿の女性カウンセラーと向かい合う女性。口は閉じたまま、話していない。安心した表情でうなずく。温かい照明。 テキストなし。
**動画化**: 口を閉じたまま、話さない。カメラがゆっくり近づく。表情が明るくなっていく。うなずきのみ(発話なし)。

### シーン6(保存名: scene_06.mp4 / 尺目安10秒)
**画像**: 白のワンピースを着てカメラを見て柔らかく微笑む女性。口は閉じたまま、話していない。明るい自然光。シンプルな背景。 テキストなし。
**動画化**: 口を閉じたまま、話さない。カメラ固定。女性が微かにうなずく(発話なし)。温かい光。
