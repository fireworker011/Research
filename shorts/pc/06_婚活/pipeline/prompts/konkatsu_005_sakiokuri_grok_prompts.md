# konkatsu_005_sakiokuri — Grokアプリ用プロンプト一覧(ステップ実行版)

各シーンごとに、①画像プロンプトをGrokに貼って画像生成 →
②できた画像を選んで動画化ボタン →③動きの指示(動画プロンプト)を貼る。
完成したらファイル名を `scene_01.mp4`〜`scene_06.mp4` にして
このチャットに送る(またはinput/フォルダに配置)。

---

## シーン1 (目安10秒)

**① 画像プロンプト(コピペ用)**
```
30代前半の日本人女性、茶色がかった黒髪ミディアム、細面、清楚感のある顔立ち。ベージュのニット。縦長9:16のリアルな写真風。 自宅のソファでカメラに向かって真剣な表情で語りかける女性。少し後悔のにじむ表情。柔らかい室内照明。 テキストなし。
```

**② 動画化の動きの指示(コピペ用)**
```
カメラがゆっくり近づく。女性の表情にフォーカス。
```
