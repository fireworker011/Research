# konkatsu_005_sakiokuri — Grokアプリ用プロンプト一覧

作る素材は**6本だけ**(各10秒前後でOK)。パイプラインが12カットに割って高テンポ化します。
各シーン: ①画像プロンプトで画像生成 → ②動画化 → ③動きの指示を貼る。
完成したら `scene_01.mp4`〜`scene_06.mp4` の名前でこのチャットに送ってください。

【全シーン共通・毎回プロンプトの先頭に貼る】
```
30代前半の日本人女性、茶色がかった黒髪ミディアム、細面、清楚感のある顔立ち。ベージュのニット。縦長9:16のリアルな写真風。
```

## scene_01 (フック: 語りかけ)

**① 画像プロンプト**

自宅のソファでカメラに向かって真剣な表情で語りかける女性。少し後悔のにじむ表情。柔らかい室内照明。テキストなし。
